本页所有内容是东方归言录的语录。

# 注意事项

  • 以下绘礼均 满突破、满强化 数值。
  • 会心攻击 = 暴击 ； 会心命中 = 暴击率
  • 阳 / 阴攻一等级上升 30% 伤害
  • 展示的绘礼均能满破能用，不用着急分解。再实用的符卡也有上位

# 绘礼 - 梅

烈日下的不融的冰妖精

# 烈日下的不融的冰妖精 ☆☆☆☆

  • 体力 + 275
  • 阴防 + 85

# 解说

  • 解说①
    • 琪露诺
    • 在雾之湖附近出没的冰精。
    • 不要以为她是冰之妖精，夏天就无法发挥本事了…… 这么想肯定会被冻伤的，决不可以大意。
  • 解说②
    • 大妖精
    • 经常和琪露诺一起行动的妖精。
    • 和在妖精当中最强 (?) 的琪露诺时常在一起，大妖精某种意义上可能也是最强吧。
  • 解说③
    • 幻想乡也迎来了夏天！
    • 受不了炎热的琪露诺在寻找避暑的地方。在没有海的幻想乡，小溪是最有人气的地方。如果去 “玄武涧” 的话，就会看到很多妖精妖怪在乘凉。“这里就由我收下了！” 随着突然的叫喊声．冻符 “完美冻结” 炸裂了开来。
  • 解说④
    • 看到这冻得硬邦邦的河川，大酱一定会大惊失色吧。在这炎炎夏日使用着能力的琪露诺，原本是一边漫无目的地溜达一边在结冰的河面上纳凉但是...... 突然，被冻住的河童们大发雷始！琪露诺该怎么办！？
  • 解说⑤
    • 因为住所被冻起来而生气的荷取。
    • 在荷取特制钻头爆发的火花下，小溪的冰被敲碎了…… 只见琪露诺横躺在上面，看来是不小心跌进了小溪里的。急着从小溪中出来的琪露诺一不小心把小溪冻住了！即使有大妖精的帮助也花了很大功夫才脱身…… 加油，大妖精！

# 特性

  • 装备符卡使用时阳攻上升 2 等级 (2 回合)
  • 装备符卡的火属性的威力上升 10%(1 回合)

# 绘礼展示

夜间寺子屋

# 夜间寺子屋 ☆☆☆☆

  • 阳防 + 60
  • 阴防 + 80

# 解说

  • 解说①
    • 琪露诺
    • 她是住在湖附近的笨 —— 冰之妖精。
    • “因为感觉被人当作笨蛋”，她频繁地前来参加授课，但没一次看见她拿起铅笔写字。
  • 解说②
    • 大妖精
    • 勤快地照顾琪露诺 —— 或是说，经常跟她一起玩的妖精。
    • 和往常一样，这次她也作为琪露诺的监视人兼保护者一同参加授课，但她忘了那份职责，集中精神学习了起来。
  • 解说③
    • 露米娅
    • 她是潜藏在黑喑中的妖怪。
    • “琪露诺要上课的话我也要。” 虽然她这样说着意气扬扬地出席了授课，却在试卷上写完名字之后就睡着了。或许她在梦中好好学习 ——“可以吃吗……? ”
  • 解说④
    • 上白泽慧音
    • 她是在人类村落担任学堂教师的半兽半人。以前在《文文。新闻》的采访中透露过她曾在考虑办历史学校，这个想法实际成型的样子便是这间 “学堂”。也教授语文和算术。
    • 白天面向人类的孩子，夜间面向妖精和妖怪招募希望入学者进行授课。
  • 解说⑤
    • 闭校时间不断接近，慧音看了看整个教室，今晚也令她十分烦恼。
    • 先不说一个劲儿学习的大妖精，要让琪露诺和露米娅正确地学到知识，该留怎样的作业才对呢。
    • 不管怎样，先从用惩罚头槌把这次也自始至终一直在睡觉的露米娅叫起来开始吧。就这样慧音踏出了脚步。

# 特性

  • 装备符卡的灵力回收效率上升 50%(1 回合)
  • 使用时自身灵力上升 1.25

# 绘礼 - 兰

服装店·香霖堂

# 服装店・香霖堂 ☆☆☆☆

  • 体力 + 315
  • 阳攻 + 77

# 解说

  • 解说①
    • 铃仙二号
    • 居住在月亮的玉兔。
    • 平时都是和丰姬和依姬一同行动的，但这次却一个人来地面玩了。对此，她是这么说的：偶尔也想要好好休息放松一下！。”
  • 解说②
    • 因幡帝
    • 地上的兔子。因为一直都很注意健康，所以变得十分长寿。平时都住在永远亭里，经常捉弄铃仙。今天该用什么恶作剧好呢，帝的脸上露出了不会好意的微笑 ———”
  • 解说③
    • 铃仙・优县华院・因幡
    • 原・月兔。时不时会带着在永远亭里制作的药去人类的村落贩卖。
    • 带着从月亮来玩的铃仙二号去迷途竹林的时候...…… 漂亮的掉进了陷阱里去。铃仙的怒吼，响彻了整座迷途竹林。
  • 解说④
    • 因为因幡帝的恶作剧而掉入了落穴的铃仙。因为衣服被划破，没办法只能去香霖堂买新衣服了。
    • “哎 —— 穿件和以前的衣服不一样的不好吗？” 帝偷看着试衣间说道。
    • “还是穿以前的更方便行动…… 话说，不要偷看啦！” 铃仙慌忙地说道。
    • 在一旁看着两人的铃仙二号激动地感叹着 “这就是地面的制服啊～”。以及守望着两人的购物的霖之助。
  • 解说⑤
    • 森近霖之助
    • 是香霖堂的店主。在位于魔法森林的入口处，经营着一间道具店。
    • 过去曾在有名的道具屋，雾雨家里工作。虽说那个时候提出了 “这里并不能让自己的能力最大限度发挥出来” 的独立言论，可实际上的原因… 恐怕也只有本人清楚了。”

# 特性

  • 装备符卡的日属性的威力上升 20%(1 回合)
  • 装备符卡使用时受到来自通常弹的伤害下降 15%(3 回合)

# 绘礼展示

春天来啦~

# 春天来啦～☆☆☆☆

  • 阳攻 + 72
  • 阳防 + 68

# 解说

  • 解说①
    • 莉莉白
    • 带来春天的妖精。又名，春告精。通过找到常人不会注意到的细微变化，来告诉冬眠中的生物和幽灵们春天已经到来。
  • 解说②
    • 如果能遇到传递春天到来的莉莉的话，说不定可以比其他人更先感受到春天的到来哦。
  • 解说③
    • 可能因为莉莉太专注于告知春天到来的消息了，所以有一些天然呆……
    • 似乎有时会被人类的孩子抓住，有时又差点被咲夜装到瓶子里去。
  • 解说④
    • “莉莉同爱塔妮缇拉尔瓦的春夏对决，还有蕾蒂的春冬对决大概不知道在哪就进行过了。这样一想的话，莉莉貌似都没有赢过...... 要说为什么，在挑战来捣乱的桑尼时貌似她也被打败了。”
  • 解说⑤
    • 比不论发生什么事件总会第一个赶到现场的文文更先发现春天的莉莉白，在收集事件信息方面也十分擅长 (只和春天相关)
    • 在另一个幻想乡里，莉莉白作为宣传委员大显身手，大概会和文文进行情报对决也说不定。但两边都缺乏可信程度，似乎分胜负就用有趣程度来决定了。

# 特性

  • 装备符卡使用时灵力上升 1.5 (1 回合)
  • 装备符卡使用时受到来自春的伤害下降 15%(3 回合)

# 绘礼 - 竹

深夜红茶

# 深夜红茶 ☆☆☆☆

  • 阳攻 + 64
  • 阴攻 + 76

# 解说

  • 解说①
    • 蕾米莉亚・斯卡雷特
    • 红色恶魔，居住在红魔馆的吸血鬼。吸血鬼正如其名是 “吸取血液” 的生物。红茶的红色也让人不禁联想到血液。
  • 解说②
    • 尚且不明斯卡雷特家是起源自哪个国家的。但是考虑到其吸血鬼的血统，应该有不少人都想到了东欧吧。
  • 解说③
    • 从斯卡雷特姓的发音和雇用女仆和华人这几点来看，让人联想到英国贵族。
    • 红茶也是要素之一。要怎么想是随便啦，但建议不要试图去探寻斯卡雷特家的根源。会被掌管时间的银色飞刀给刺穿身体的。
  • 解说④
    • “如果你误入红魔馆，首先应该去问候身为主人的蕾米莉亚。彬彬有礼之人和无礼之徒，那边能活得更久，就算不去细想聪明的你也应该能清楚吧......”
  • 解说⑤
    • “在被她中意的时候，大概会告诉你她喜欢的红茶种类什么的。“夜晚还长现在才开始，好好享受吧” 若是被这样邀请的话…… 只要没有急事就随她一同过去吧。”

# 特性

  • 装备符卡使用时灵力上升 1.5 (1 回合)
  • 装备符卡使用时受到来自女仆的伤害下降 15%(3 回合)

# 绘礼展示

红胜还是白胜？

# 红胜还是白胜？ ☆☆☆☆

  • 体力 + 425
  • 速度 + 55

# 解说

  • 解说①
    • 小恶魔
    • 负责整理红魔馆大图书馆里的书籍，担任着图书管理员一职。没有正式的名字，仅因为外貌就被称作 “小恶魔” 了。平时的话，会帮帕秋莉打打杂或者在身边照顾她。
  • 解说②
    • 红美玲
    • 红魔馆的门卫。经常有站着睡着的情况（本人极力强调是醒着的）。
    • 运动之秋！于是 “因为是运动之秋，咱们举办运动会吧！蕾米莉亚十分唐突地说了出来 ——
  • 解说③
    • 帕秋莉・诺蕾姬
    • 居住在红魔馆的大图书馆里的魔法使。
    • 被朋友蕾米莉亚问到 “运动会上都会比什么呢？”，于是拿出了许多书籍，向蕾米莉亚说明了运动会上的各个项目。为什么会问这个呢？虽然觉得挺不可思议的但还是继续向解释着的帕秋莉，之后马上就会后悔自己现在的行为。
  • 解说④
    • 红方代表，蕾米莉亚夺下第一，而白方代表 —— 灵梦和魔理沙还在努力追赶。“雾之湖” 周围开始骚动起来，闻讯而来的灵梦和魔理沙也被卷入其中。赛道赛跑和障碍物赛跑，投球比赛，传统骑马战，还有拔河和两人一足的项目也没有落下。
  • 解说⑤
    • 红队，处于优势 —— 芙兰破坏了障碍物！在障碍物赛跑中获得一等奖！咲夜停止时间，直接将大球搬到终点获得第一名。
    • 白队，再加油吧 —— 在骑马战因为铃仙的红眼而发狂自灭，多亏翠香和勇仪的努力在拔河中获胜，两人甚至都没有让出一步。
    • 最终项目，将在接力赛跑中一决胜负 —— 最后一棒的蕾米莉亚和灵梦正在互相瞪眼。

# 特性

  • 装备符卡的动能弹的威力上升 20%(1 回合)
  • 装备符卡使用时，受到来自红魔馆住民的伤害下降 20%(3 回合)

咱好恨啊~

# 咱好恨啊～☆☆☆☆

  • 阳攻 + 68
  • 阴攻 + 72

# 解说

  • 解说①
    • 多多良小伞
    • 无人使用的伞化作的附丧神。
    • 也即是所谓唐伞妖怪。不管是什么道具都寄宿着神灵，绝对不可以不当一回事。
  • 解说②
    • “附丧神” 是指 —— 长时间使用的道具，如果没有供养而拾弃的话，而无法回到原本的神明那里的神。也有可能会去对原本的持有者干坏事。
  • 解说③
    • 小伞是个很会照顾孩子的妖怪。如果有哭泣的孩子她便会急忙赶过去逗乐，如果有高兴的孩子她又会大声吓唬（没办法这是妖怪的本能)。被大人所讨厌的她，因为设计新奇的唐伞在孩子里面颇具人气（本人似乎有些不满）。
  • 解说④
    • 小伞虽然已经融入了命莲寺，但其实并没有拜师入门。小伞在这里的理由是，命莲寺有个人类经常造访的墓地，她为了方便吓人才经常待在这里。
  • 解说⑤
    • “二月八日是供奉那古老的银针的日子。小伞在这一天要是得不到新工作的话，就没法愉快地玩过这一年。以前，灵梦的银针曾被重新制作过，那个时候由于质量检查的确认而会被攻击。结果的话当然是，合格的。”

# 特性

  • 装备符卡使用时阴攻上升 2 等级 (2 回合)
  • 装备符卡使用时受到来自水属性的伤害下降 25%(2 回合)
