# 烈日下的不融的冰妖精 ☆☆☆☆

  • 体力 + 275
  • 阴防 + 85

# 解说

  • 解说①
    • 琪露诺
    • 在雾之湖附近出没的冰精。
    • 不要以为她是冰之妖精，夏天就无法发挥本事了…… 这么想肯定会被冻伤的，决不可以大意。
  • 解说②
    • 大妖精
    • 经常和琪露诺一起行动的妖精。
    • 和在妖精当中最强 (?) 的琪露诺时常在一起，大妖精某种意义上可能也是最强吧。
  • 解说③
    • 幻想乡也迎来了夏天！
    • 受不了炎热的琪露诺在寻找避暑的地方。在没有海的幻想乡，小溪是最有人气的地方。如果去 “玄武涧” 的话，就会看到很多妖精妖怪在乘凉。“这里就由我收下了！” 随着突然的叫喊声．冻符 “完美冻结” 炸裂了开来。
  • 解说④
    • 看到这冻得硬邦邦的河川，大酱一定会大惊失色吧。在这炎炎夏日使用着能力的琪露诺，原本是一边漫无目的地溜达一边在结冰的河面上纳凉但是...... 突然，被冻住的河童们大发雷始！琪露诺该怎么办！？
  • 解说⑤
    • 因为住所被冻起来而生气的荷取。
    • 在荷取特制钻头爆发的火花下，小溪的冰被敲碎了…… 只见琪露诺横躺在上面，看来是不小心跌进了小溪里的。急着从小溪中出来的琪露诺一不小心把小溪冻住了！即使有大妖精的帮助也花了很大功夫才脱身…… 加油，大妖精！

# 特性

  • 装备符卡使用时阳攻上升 2 等级 (2 回合)
  • 装备符卡的火属性的威力上升 10%(1 回合)

    • 勤快地照顾琪露诺 —— 或是说，经常跟她一起玩的妖精。
    • 和往常一样，这次她也作为琪露诺的监视人兼保护者一同参加授课，但她忘了那份职责，集中精神学习了起来。
  • 解说③

    • 她是潜藏在黑喑中的妖怪。
    • “琪露诺要上课的话我也要。” 虽然她这样说着意气扬扬地出席了授课，却在试卷上写完名字之后就睡着了。或许她在梦中好好学习 ——“可以吃吗……? ”

    • 上白泽慧音
    • 她是在人类村落担任学堂教师的半兽半人。以前在《文文。新闻》的采访中透露过她曾在考虑办历史学校，这个想法实际成型的样子便是这间 “学堂”。也教授语文和算术。
    • 白天面向人类的孩子，夜间面向妖精和妖怪招募希望入学者进行授课。
  • 解说⑤
    • 闭校时间不断接近，慧音看了看整个教室，今晚也令她十分烦恼。
    • 先不说一个劲儿学习的大妖精，要让琪露诺和露米娅正确地学到知识，该留怎样的作业才对呢。
    • 不管怎样，先从用惩罚头槌把这次也自始至终一直在睡觉的露米娅叫起来开始吧。就这样慧音踏出了脚步。

暖好被炉等着你

# 暖好被炉等着你 ☆☆☆☆

  • 体力 + 250
  • 阳防 + 90

# 解说

  • 解说①
    • 博丽灵梦
    • 在寒冷冬日缺不了热茶的博丽巫女。
    • 今年元旦来了很多参拜客十分满足。坐进被炉，喝杯热茶…… 你是如何度过正月的呢？
  • 解说②
    • 雾雨魔理沙
    • 虽然迷你八卦炉也很好，但还是被炉最棒的普通的魔法使。
    • 今年参拜客很多，她和阿哗一起赶来帮忙。在帮完忙后，当然要坐进被炉里吃橘子。你吃了几个橘子呢？
  • 解说③
    • 关于 “被炉”—— 有种说法是，被炉四周的框架部分因为和下牛车时使用的叫做榻的踏脚台很相似，所以就写作 “火榻子” 读作 “kotatsu”。它由室町时代的禅僧们传播开来，炬缝和火燧是假借字。
  • 解说④
    • 在被炉里暖着身子吃着橘子，突然魔理沙说出了 “为什么被炉和橘子会搭在一起啊？” 这个疑问。冬天里橘子在常温下能保存两到三周。另外，吃着很方便，徒手就能剥皮。而且，还富含维生素 C 等营养，也能补充水分。灵梦并没有说明这些，只是说了句 “因为好吃吧” 打发了过去。
  • 解说⑤
    • 现在一边在被炉里暖着身子，一边思考着抱负与今后的封结晶回收及失言异变解决对策。小心不要感冒……

# 特性

  • 使用装备符卡时， 阳攻将上升 2 等级 (2 回合)
  • 使用装备符卡时， 阳防将上升 2 等级 (2 回合)

是动物灵做的！？

# 是动物灵做的！？ ☆☆☆☆

  • 速度 + 55
  • 阳攻 + 85

# 解说

  • 解说①
    • 帕秋莉・诺蕾姬
    • 她是蕾米莉亚的朋友，基本杲在红魔馆的图书馆（说她一天的大半部分都在那里度过也不过分）。
    • 在她如往常一样读书的时候，正在收拾的小恶魔拿来了本稀奇的书。
  • 解说②
    • 小恶魔
    • 她是作为帕秋莉的使魔努力工作的恶魔。另外，她的名字并非正式名称，是根据她的容貌起的。
    • 在她进行可说是每日必做工作的图书馆整理时，发现了稀奇的书。正在她准备报告给帕秋莉，向她冲过去的时候 —— 书开始施放强烈的光芒。
  • 解说③
    • 所谓 “图书馆”—— 是主要进行资料等的收集・整理・保管，供公众利用的设施。最近，也会放置漫画和电脑之类的。
    • 至于帕秋莉管理的图书馆，因位于红魔馆地下，所以只有蕾米莉亚和咲夜等少数人踏足（魔理沙时常会来玩）。
  • 解说④
    • 小恶魔拿出的书，是封印着 “某个灵” 的。封印顺着势头解开了，之前被封印的灵闹了起来！
    • 通过帕秋莉和小恶魔的奋战，成功再次封印了…… 但二人的样子有些奇怪。
  • 解说⑤
    • 听到动静赶过来的咲夜和美铃所看到的 —— 竟然是长出了兽耳和尾巴的帕秋莉与小恶魔。根据帕秋莉所说，在封印那些灵的时候它们垂死挣扎，施加了诅咒。还说过几天就会恢复…… 但在恢复原样之前据说有很多人造访了图书馆。

  • 装备符卡的光弹威力上升 20%(1 回合)
  • 使用时自身受到来自猫的伤害下降 20%(3 回合)

# 青蛙帽子 ☆☆☆☆

  • 阳攻 + 70

# 解说

  • 解说①
    • 泄矢诹访子
    • 土著之神。也是将御社宫司神所管束着的神。所谓的 “御社宫司神”—— 是负责 “出生”“农业”“军事” 这些各种各样的作祟神。若是稍有轻视，便会降下神罚将其惩戒的恐怖之神。守矢神社的真正的祭神。并不是蛙之神。
  • 解说②
    • 尽管她的口吻毫无威严，外表和神明这一称谓相比太过幼小，还有她那懒懒散散的态度…… 也许反而如实反映了这个国家的神性吧。
  • 解说③
    • 蛇克蛙，被蛞蝓克。蛙克蛞蝓，被蛇克。蛞蝓克蛇，被蛙克。
  • 解说④
    • 蛇、蛙和蛞蝓，就是世间常说的 “三极”。毫无疑问是和赤口大神的信仰有某种关系。
  • 解说⑤
    • 原住神被新神所入侵，那信仰的根基被夺走这一历史遗留，在这个国家是随处可见的。并不是所谓的 “宗教改革”, 而是变迁本身，被各种各样的策略所反复使用才逐渐去确立的。
    • 那就好比，大蛇紧紧缠绕在猎物身上逐步吞食一般的过程吧。

# 绘礼 - 竹

大小姐与我
  • 阳攻 + 55
  • 阴攻 + 85

# 解说

  • 解说①
    • 【蕾米莉亚・斯卡雷特】
    • 度过了五百年以上的漫长岁月的吸血鬼。在幻想乡中筑起红魔馆，与妹妹、友人和许多仆从一同居住。
  • 解说②
    • 所谓的 “吸血鬼”—— 因种种虚构作品而广为人知的吸食活人血液的恶魔。
    • 不同作品中的解释略有不同，但对于 “弱点是阳光和银器” 这类种族特征的描述基本一致。蕾米莉亚本人也和作品中的描述大致相符。不，其实因果恰恰相反。应当说，是曾与她们遭遇的人出于畏惧而创作了那些作品，其中记述了她们的特征。
  • 解说③
    • 惹人怜爱的幼小女孩。举手投足之间流露的优雅可以印证她良好的家教。
    • 然而若是亲眼直视过她本人，一定能感受到某种无法用言语传达的东西。小小的身躯内部散发出的绝非等闲之辈的气息。无法掩饰的某种气场。这也可以说是一种领袖气质吧。
  • 解说④
    • 寄身于馆中的人们出身、种族、思绪都各不相同。即便如此，仍有唯一一条共识让她们团结在一起共同战斗。为了她的身姿。她的心灵。她的力量。或是她的任何魅力。构成她的一切 —— 这一切的集合体正是蕾米莉亚・斯卡雷特，这位将众人深深吸引的馆主。
  • 解说⑤
    • 这恐怕是每天 —— 不，是一天之内也会反复目击到的光景吧。她一步一步地踏着台阶，款款走下楼梯。仅此而已。
    • 对她的身姿投以短暂的凝视后，在楼梯下等候的红魔馆女仆长静静低下头去。此情此景并非是什么理所当然。女仆长的眼神仿佛在诉说 ——“她的身姿已然宣告了一切”。

# 特性

  • 使用时敌方单体阴防下降 2 等级 (3 回合)
  • 使用时自身受到来自吸血鬼的伤害下降 25%(2 回合)

# 一如既往的赛钱箱 ☆☆☆☆

  • 体力 + 350
  • 阴攻 + 70

# 解说

  • 解说①
    • 博丽灵梦
    • 博丽神社的巫女。和往常一样进行着打扫，就好像不管参拜客什么时候会来都提前做好准备一样，但...... 一如既往的，根本没有人来。到访的尽是前来看望灵梦的妖怪和熟人们。
  • 解说②
    • 在使用了千方百计后，试验一下会不会有参拜客来什么的…… 实际上一般人在听到 “博丽神社有妖怪出没” 的传闻之后，当然更不敢靠近了。对于这件事，灵梦有所察觉到吗......
  • 解说③
    • 有一次，赛饯箱被塞满了。灵梦开开心心打开来一看…… 结果里面被塞满了树叶。当知道是妖怪狸、妖怪狐或者是妖精们的恶作剧的时候，灵梦开始做出门的准备。当然，是去妖怪退治了。
  • 解说④
    • 如果你不小心进入了幻想乡的话，那就去博丽神社吧。
    • 相比以灵梦为首的很多人都会助你一臂之力的吧。
  • 解说⑤
    • 给灵梦打下手的时候，记得要投些赛钱。
    • 这么做了的话，她会十分乐意给你 “祈祷” 的哦。愿意帮助你的伙伴也会增加的吧。

  • 装备符卡的无属性的威力上升 10%(1 回合)
