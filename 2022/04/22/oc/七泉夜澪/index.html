# 虚灵

# 设定

  • 名前：七泉夜澪(ななみやみお)
  • 年龄：3 岁 / 18 岁 (外表)
  • 生日：1.24
  • 体重：52kg
    • 0≤幽灵状态＜15kg
  • 胸围：B
  • 种族：幽灵
  • 象征物：黑蔷薇
  • 身高：173cm/179cm（鞋跟）
  • 武器：镰刀
  • 能力：死亡引导与死亡操纵 / 操纵金属
    • 可操纵行尸走肉的尸体（需灵魂）
    • 左眼「归寂」：大范围强化能力使用限制，可引导灵魂（的容器）。过度使用后左眼会流血，且短暂失明
  • 别名：小暗、夜夜
  • 有极微小的容器力量
  • 可自由来往彼岸，看得见死灵
  • 因家族原因，很有钱（富婆）
  • 喜欢黑色的东西
  • 很单纯，容易被骗。（遗传？）

# 人际关系

  • 喜欢的人：鬼凛堂桜、七泉惠之夏
  • 亲人：七泉明一、七泉兰、界、音
  • 朋友：决星呱、芙利奇亚、弥一百依、绯上灵琰、御守院馨

# 简介

  • 在家主七泉明一的愿望之下，以及墓地中的各种怨念所诞生的幽灵，相貌随【七泉】家中的女儿 - 惠之夏。
  • 因七泉明一的愿望不够强烈，导致无法让真正的七泉惠之夏复活，取而代之，以幽灵形式诞生的夜澪被七泉明一收养，当成女儿养育.(相貌很像)
  • 左右眼是不同的十字架，左眼的十字架拥有操纵死亡的能力，平时都是用绷带藏起来的。能与幽灵进行对话，看得见寿命长短.
  • 虚灵与死灵的区别在，虚灵不会因怨念而失控，生前死的时候也没有怨念。
  • 即使不用左眼都可以使用【死亡引导】与镰刀，以及一些小规模的能力，所以实力十分强悍。
  • 在七泉家中时，很多时候都与灵魂状态的惠之夏进行对话，过着惠之夏想要的生活，与人类一起生活的时光。
  • 称鬼凛堂桜为【妈妈】 是母控
  • 经常谎称自己要去旅行，其实是去找妈妈玩
  • 性格很活泼开朗，是个交际花。
  • 清楚自己和惠之夏只能存在一个。
  • 性取向正常！
  • 在学校里很受欢迎
    • 有女生讨厌她的话，她会尽量解除误会
    • 即使有表白的男女生，也会很礼貌的拒绝
  • 愿意帮助任何人

# 能力解析

  • 有两种能力：能够操纵金属 / 拥有操纵死亡的能力
  • 操纵金属的能力：身边半径 5 米内的 金属元素 可以在夜澪手里化作任何形态的变化。
    • 左眼「归寂」能力发动后，能力范围扩大可至半径 200 米。
    • 0~50 米时，左眼会刺痛
    • 50~100 米时，左眼视力暂时下降
    • 100~200 米时，左眼流血，且短暂失明
  • 拥有操纵死亡的能力：最多同时操作两人的生命值，生命变化。灵魂操作则只能操作个单位。
    • 左眼「归寂」能力发动后，能力范围扩大可至半径 1 千米
    • 一旦使用，左眼流血，且短暂失明。
    • 强化能力，能引导灵魂。
  • 定位是「妨碍 / 干扰」
