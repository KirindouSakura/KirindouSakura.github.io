# 绘礼 - 梅

灵异社团

# 灵异社团 ☆☆☆☆☆

  • 阳防 + 76
  • 阴防 + 74

# 解说

  • 解说①
    • 《在某个「膜」的某个「断面」上。好像有类似这样的话。》
    • 「最近怎么样，梅莉」
      • 「迟到了 1 分 20 秒」
      • 「是吗，赶上了呢」
      • 「完全没有赶上」
    • 莲子想要知道。姑且算是有痊愈的迹象了。那只是没有被他们的眼睛观测而已。莲子无论是过去还是现在。一直在观察梅莉异常而疯狂的能力。
    • 「和平时有点不同的活动呢」
      • 「和往常一样，到处多看看吧」「我并不讨厌到处转悠」
      • 「能满足公主的希望，荣幸之至」
    • 那么今天也。—— 开始秘封俱乐部吧。
  • 解说②
    • 宇佐见莲子
      • 灵异社团「秘封俱乐部」的成员。被人叫做「莲子」。
      • 在自己知道的人中，拥有最高的头脑。[M.H. 氏谈]
    • 玛艾露贝莉・赫恩
      • 灵异社团「秘封俱乐部」的成员。被人叫做「梅莉」。
      • 在自己知道的人中，最有神秘感。[R.U. 氏谈]
  • 解说③
    • 古代物理学迎来了终焉。电磁气力、弱力、强力，还有重力。已经被证明，所有的物理现象，只不过是万物的理论「超统一物理学」的一面而已。那是叫做实用向超弦理论，脑力世界……「膜宇宙」等概念，揭开我们世界尊颜的瞬间。
    • 作为现代人，那不过是一种「教诲」。物理学者们只是为神官，翻译从大宇宙中传来的神谕而已。然后他们也没有「单独的」了解到宇宙的原貌。所以至今还能看到统一外的「灵」。不同的人以不同的方式看到。灵能……「灵力之场」，就是指那个吧？
    • 那是根据每个观测者而变化的宇宙。不只是四个，把实际上无数存在的宇宙统一起来，这可能吗？莲子和梅莉一边聊一边想象着。
  • 解说④
    • 秘封俱乐部
      • 莲子和梅莉，在大学期间的社团活动。
    • 虽然是灵能者社团，但是像这种类似的灵异社团的降灵会或者除灵活动都并没有做过。
    • 所以在其他的灵能大学生眼里，她们是在霸占教室与预算，的的确确是不良社团的样子。
    • 那么，她们到底做些什么呢。
    • 其实，她们在通过结界和梦境「到异世界中冒险去了」。
    • 还有就是…… 把她们去异世界的体验做成同人志之类的？
  • 解说⑤
    • 医生的诊断是「病毒性儋妄」。
      • 赫恩家族传承的「寄宿于血脉的灵能」。莲子觉得无论哪个「都说的通」。但是这离「唯一的真理」还差的很远。
      • 让她察觉到这一点的就是玛艾露贝莉・赫恩。对于莲子来说的梅莉。常常是作为魔术师的梅莉。
    • 「很亲切的老师呢。帮我找到了病历卡，时间晚了，还特地用亲戚的住宿招待我们」
      • 「在我们蒙受关照的时候也一直非常亲切热情」
      • 清晨的信州。
      • 空气清新，心情凉爽。
    • 「那么，关于昨天晚上就很在意这个地图。」莲子这么说的时候。梅莉已经把裙子下摆撩起…… 正在跨过栅栏。哎呀哎呀，真是个了不起的公主啊。

  • 装备符卡的镭射弹威力上升 35%(1 回合)
  • 使用时自身回避上升 2 等级 (3 回合)

# 绘礼展示

天空的秘神

# 天空的秘神 ☆☆☆☆☆

  • 阳防 + 75
  • 阴防 + 75

# 解说

  • 解说①
    • “确实是门没错…… 但没想到我会在这种地方担任 “门卫” 呢” 隐岐奈一边叹气一边说道。
    • 一定是相当有地位的人吧……? ,“.…… 到底是什么人呢？” 在后方待命的舞女们纳闷道。
    • “你们不用在意这种事情”—— 我可不能把你们卷进来。隐岐奈在心里说道。
  • 解说②
    • “贤者” 是指贤明之士的词语。
      • 根据时代以及地区也会指为国王出谋划策的智者以及学者。
      • 反义词是 “愚者”
      • 现在的我，简直就是愚者。
    • “幽明结界”。
      • 隔离冥界与幻想乡的大结界。
      • 应当守护这里的人并不是我…… 不，这已经无所谓了。
  • 解说③
    • “里乃。舞。你们没有必要一直陪着我”“就算您这么说”
    • “我们也不能让您一个人啊” 当然并非一两个小时而已。
    • 等待的时间早已超出了人类的感觉能够测量的时间。
    • “既然如此，我就给你们讲讲以前的故事吧” 真是的，到底是指 “哪个” 以前啊……… 包括这句回复在内，都是固定的套路。
  • 解说④
    • 我曾经侍奉与某个高贵之人。
    • 我自称来自大陆的贤者一族，将各式各样的智慧授予了君主。
    • 特别是关于艺能，那位君主十分出众。
    • 无论是在那之前还是之后都没有人编写过如此震撼人心的剧目。
    • ………… 不知道这份记忆是我原本就有的记忆。还是从哪里混进来的记忆呢。
  • 解说⑤
    • “差不多到时间了”
      “终于要到时间了”
      二童子兴奋了起来。
    • “还有些时间。你们先冷静一下。”
    • 巫女就要来了。
    • 如果是有问题的巫女的话，矫正巫女的人无论何时都是身为 “神中之神” 的我。
      “因为这可是究极的绝对秘神，摩多罗隐岐奈的登场啊”

  • 装备符卡的动能弹威力上升 50%(1 回合)
  • 使用时自身会心攻击上升 2 等级 (1 回合)

# 绘礼 - 竹

幽玄的剑圣

# 幽玄的剑圣 ☆☆☆☆☆

  • 阳攻 + 75
  • 阴攻 + 75

# 解说

  • 解说①
    • 魂魄妖梦
      • 人类幽灵各占一半的少女。虽说是一半但却并不是混血，而是被称为 “半人半灵” 的种族。在位于冥界的白玉楼里，是作为服侍主人西行导幽幽子的从者。虽说是剑术指导和兼职艺苑师，但剑道还尚未成熟。
      • 白玉楼的剑术指导，并兼任园艺师的少女。也是以幽幽子为主人的从者。
      • 妖梦虽然在各方面都很半吊子，却让人恨不起来，而且反倒是惹人怜爱的努力家…… 本该是这样，但平行世界中遇到的妖梦似乎 “有些不同”。
  • 解说②
    • “幽幽子大人。大盘子我来拿。"
      “没事的啊，妖梦。就这点儿东西。”
      “啊啊，危险！”
      “哎呀！”
    • 有点儿不同……? 不，或许是基本相同。至少，在日常生活方面是这样。
  • 解说③
    •  咚——————

    •        咚————————
      • 像是钟声一样令人毛骨悚然的声音回响起来。是” 刻 “。
      • 是从西行妖…… 那棵巨大又古老的樱花树那边传来的。听到那声音的人，都会被吸引过去。
  • 解说④
    • 白玉擂台
      • 位于平行世界白玉楼的巨大武斗台。在这里进行的决斗被称为 “比赛”。
      • 幽幽子、妖梦、蓝和橙管理着这个” 比赛 “…… 但，说到底是谁最开始制定这规则的呢？
  • 解说⑤
    • “…… 那么”
      “刻” 之音响起，身着黑色装束的妖梦端正起姿势。
    • 到该前往擂台的时候了。
    • 这是规则，是宿命，是约定，是祈祷 ————
      —————— 是诅咒。
