# 绘礼 - 兰

上镜的少女们

# 上镜的少女们 ☆☆☆☆☆

  • 阳攻 + 85
  • 阳防 + 65

# 解说

  • 解说①
    • 风见幽香
      • 她是四季的花之啦啦队队长。
      • 在一年之始的春季，成立了为无精打采的幻想乡送去精力的五人伴奏 “报春五人组”。
  • 解说②
    • 小野琢小町
      • 她是引出精力的善倾听者。
      • 她在三途之河上以船夫为职，擅长倾听对方讲话。据说她很期待能听到和渡往彼岸的幽灵所讲述的有所不同的故事。与小町聊过的人，会神奇地恢复精神。这在某种意义上，或许也是成佛。
  • 解说③
    • 依神紫苑
      • 她是快支付精力的账单。
      • 或许是因为大家都无精打采，饭也吃不下了。孙子心便当滞销，发生紧急库存损耗！
      • “不吃饭就打不起精神所以快吃吧！” 紫苑向人们如此劝告（她察觉到自己吃就好了，也已经是在幻想乡恢复精力之后了)。
  • 解说④
    • 雏的助威・传说
      • 最近，三途之河超级忙。怎么了？为了探明原因（以此为名的偷懒）造访人类村落一看…… 大家都无精打采的！
      • 笑容从幻想乡中消失了。到底发生了什么呢，小町开始打听起来 ——
  • 解说⑤
    • 根据永琳所说 —— 人笑一笑就会稍微延长些寿命。那怎么可能…… 小町虽然曾经对此一笑了之，但看着现在的幻想乡，开始觉得那也未必是错的了。
    • “只要有精神就能长寿！” 曾经有谁这样说过。小町加入了五人伴奏 “报春五人组”，向大家送上声援。

# 特性

  • 使用时敌方全体会心回避下降 2 等级 (2 回合)
  • 使用时自身灵力上升 1.25

# 绘礼展示

追忆之时

# 追忆之时 ☆☆☆☆☆

  • 速度 + 85
  • 阳攻 + 95

# 解说

  • 解说①
    • 八云紫
    • 落樱不再，烟雨蒙蒙。
    • 回首间，恍然如梦，终是幻影。
  • 解说②
    • 蝉鸣扰扰，银雪绵绵。
    • 玉阁春色清梦着，千年一隙间。
  • 解说③
    • 月影杳香，靛裳翩翻。
    • 华伞旋落境界现，何处幻乡何处眠。
  • 解说④
    • “那棵樱树” 即将开花之时，春色仿佛也溜进了梦中。
    • 悠悠转醒，千年时光像弹指一瞬，从隙间轻灵地溜走一般。
  • 解说⑤
    • 越是无法拥有之物，越是心存幻想。死亡是生者的幻想。
    • 这片幽深的幻想之境，会不会也只是隙间妖怪的悠然一梦呢。

# 特性

  • 装备符卡使用时受到来自人类的伤害下降 30%(2 回合)
  • 装备符卡的尖头弹的威力上升 20%(1 回合)
  • 装备符卡使用时阳防上升 1 等级 (2 回合)

# 绘礼 - 菊

人偶解放战线

# 人偶解放战线 ☆☆☆☆☆

  • 体力 + 375
  • 阴攻 + 75

# 解说

  • 解说①
    • 爱丽丝・玛格特洛伊德
      • 人偶师和魔法使。由于爱丽丝的魔法，人偶们会像有生命一般自由行动。每个人偶彼此之间都有着自己的意志，有时也会有一边觉得麻烦一边给爱丽丝帮忙的人偶。
  • 解说②
    • 梅蒂欣・梅兰可莉
      • 诞生没多久的人偶妖怪。
      • 梅蒂欣的能力被认为是由于长期被放置在开着铃兰的无名之丘而掌握的。
  • 解说③
    • 正是因为梅蒂欣十分重视着人偶，所以从人类手中解放人偶的目标才会被她提出来吧。由于爱丽丝也非常深爱着人偶，所以两人如果相见的话一定会情投意合吧。
  • 解说④
    • 玉兔 VS 河童！大生存游戏祭典
      • 清兰、铃瑚和荷取的意见而参加了大生存游戏祭典的铃仙。为了帮助铃仙，梅蒂欣被选为了医疗兵。是因为她从平常时候就给永琳提供毒来帮忙制药的吧。
      • 从某种意义上可以说是最强的医疗兵和帮手了。
  • 解说⑤
    • 爱丽丝参加大生存游戏祭典的话，可能会通过操纵人偶来战斗的吧。然后见到这一幕的梅蒂欣肯定会说着 “一定要救那个人偶！” 然后冒着枪林弹雨冲出去的吧。爱丽丝看着比起玉兔队伍的胜利更重视人偶的梅蒂欣，应该会一边感叹着真是她的风格一边苦笑吧……

# 特性

  • 装备符卡的光弹的威力上升 25%(1 回合)
  • 装备符卡使用时受到来自水属性的伤害下降 20%(2 回合)
  • 装备符卡使用时受到来自人偶的伤害下降 15%(2 回合)

# 符卡展示

新年参拜组 其二

# 新年参拜组 其二 ☆☆☆☆☆

  • 体力 + 375
  • 阳防 + 75

# 解说

  • 解说①
    • 东风谷早苗
      • 既是从外面的世界来的人类，又是生活在守矢神社的现人神。
      • 时常到访博丽神社的她，看着正准备庙会节日的灵梦不禁想到...... 自己家的神社也得干点什么。说着就立刻行动起来。”
      • 守矢神社的风祝。
      • 穿着新年用的振袖，与神奈子和诹访子一同对今年一年进行祈愿。她的愿望，是能比灵梦解决更多异变…… 也许是这样。
  • 解说②
    • 射命丸文
      • 飞来飞去寻找新闻的鸦天狗。
        穿着新年用的振袖，从新年开始记者就超级忙碌。肯定又是去天狗们的新年聚会，又是为了收集情报飞来飞去对线人们送上新年祝福。今年也请多多关照文文。新闻。
  • 解说③
    • 关于 “新年参拜”—— 据说是由守岁这个仪式发展而来的。
    • 守岁，是指村子或一家之长到祭祀了那片地区氏神的寺社中从除夕夜里到元旦早上不睡觉地度过。要一宿不睡一直祈祷。要是不小心睡着的话，白发和皱纹就会变多。
  • 解说④
    • [辞旧迎新] 弹幕突击者们
    • 谨贺新年，众多参拜客杀到了博丽神社平常门可罗雀的神社，在正月期间也是门庭若市。灵梦对 “送礼的怪物” 许下的愿望，就要实现了！?
  • 解说⑤
    • 代替忙于接待参拜客的灵梦，你被拜饦去各处拜年，在魔理沙和早苗她们的帮助下穿梭于幻想乡。
    • 来到幻想乡之后过的第一个正月。向帮助过你的人们拜年，收压岁钱…… 希望今年也是个好年。

# 特性

  • 使用装备符卡时，受到来自巫女的伤害下降 40%(1 回合)
  • 使用装备符卡时， 命中将上升 3 等级 (1 回合)

# 绘礼展示

墨染之樱

# 墨染之樱 ☆☆☆☆☆

  • 速度 + 95
  • 阴攻 + 85

# 解说

  • 解说①
    • 西行寺幽幽子
    • 墨染的樱花盛放之时，她是否回到了梦中的尘世？
    • 昨日雪如花，今日花如雪。
  • 解说②
    • 弘川埋骨，西行永眠。
    • 白玉墨染之时，蝶舞樱落。
    • 斯人已逝，唯庭阁相伴、扇月相和。
  • 解说③
    • 浮生若梦，愿银雪垂帘，春度安枕，悠然长醉，不复轮回永劫之思。
  • 解说④
    • 埋骨于弘川，永眠在不会开花的西行妖之下。
    • 当白玉楼被墨染樱花妆点之时，幽蝶纷飞、樱瓣零落。
    • 长久陪伴已逝少女的，是静默的庭阁与皓月。
  • 解说⑤
    • “古籍里记述的那人，到底是谁呢？”
    • 少女一心解开谜题，于是春雪如白银之帘一般，冰封幻想乡。
    • 聚集于此的春度，能让她暂且沉浸于梦境吗？可以的话，就让少女停留在她美好的幻梦中，永远脱离轮回之苦吧。

# 特性

  • 装备符卡使用时受到来自白玉楼住民的伤害下降 30%(2 回合)
  • 装备符卡使用时阴防上升 2 等级 (2 回合)
  • 装备符卡使用时受到来自体术的伤害下降 10%(2 回合)

# 绘礼 - 梅

新年参拜组 其三

# 新年参拜组 其三 ☆☆☆☆☆

  • 速度 + 75
  • 阴攻 + 75

# 解说

  • 解说①
    • 魂魄妖梦
      • 人类幽灵各占一半的少女。虽说是一半但却并不是混血，而是被称为 “半人半灵” 的种族。在位于冥界的白玉楼里，是作为服侍主人西行寺幽幽子的从者。虽说是剑术指导和兼职艺苑师，但剑道还尚未成熟。”
      • 半人半灵的白玉楼园艺师兼剑术指导。
      • 穿着新年用的振袖，祈愿今年一年能够幸运…… 让她用持有的白楼剑砍一下的话，或许能有斩断迷惘的功效。作为代价，幽灵会成佛人类会出血。
  • 解说②
    • 铃仙・优昙华院・因幡
      • 拥有狂气红眼的原・月兔。
      • 穿着新年用振袖，祈愿今年一年不会生病受伤…… 努力卖永琳制作的药，干劲十足。
  • 解说③
    • “新年试笔”—— 是在年初写下一年的抱负和目标的正月惯例仪式之一。在过去，会用元旦早晨新打的水（若水）磨墨，朝向新年的吉利方位写下诗歌。
  • 解说④
    • [辞旧迎新] 弹幕突击者们
      • 被拜诋去跟前途光明的孩子们传话，让他们来博丽神社参拜，你前往了学塾。
      • 在学塾，看到了接受慧音和阿求、小铃指导写下新年试笔的孩子们还有妹红。你也来写写今年一年的抱负和目标吧？
  • 解说⑤
    • 被紫带着前去白玉楼，妖梦和清兰正在拼命做年糕。
    • 正月做年糕的由来之一，是 “固齿之仪”。最开始是想要通过加固使牙齿坚固，来变得健康。正如年龄的 “龄” 字中包含 “齿” 这个汉字，固齿之仪有着祈祷长寿的意义。好好嚼，好好吃，愿无病无灾。

# 特性

  • 使用装备符卡时，受到来自月属性的伤害下降 40%(1 回合)
  • 使用装备符卡时， 阴攻将上升 3 等级 (1 回合)

# 绘礼 - 竹

新年参拜组 其一

# 新年参拜组 其一 ☆☆☆☆☆

  • 阳攻 + 75
  • 阴防 + 75

# 解说

  • 解说①
    • 博丽灵梦
      • 博丽神社的巫女。
      • 穿着新年用的振袖，祈祷生意兴隆。对近年来偶尔到来的繁忙，在忙得团团转的同时也很高兴。
  • 解说②
    • 雾雨魔理沙
      • 住在魔法森林的普通的魔法使。
      • 让霖之助准备了新年用的振袖，冲向灵梦与你所在的博丽神社不论有多着急都要注意安全驾驶哦！
  • 解说③
    • 十六夜咲夜
      • 是在红魔馆里工作的女仆长。能用保持冷静的样子去实现大小姐蕾米莉亚的各种任性要求的完美女仆。这次的要求是，听说由美铃制作出的中华料理十分美味，决定用 “即便是我也能够做到那种地步！” 来露一手的唉夜，开始了修行之旅。”
      • 完美潇洒的从者。
      • 穿着新年用的振袖，为了祈愿红魔馆的家内安全前往博丽神社参拜。祈祷可以平安无事地度过。
  • 解说④
    • 自卫星而来的压岁弹
    • 正月…… 据说掌管新年的神明，在元旦里会来到家中带来一年的幸福与健康。当然不只是地上的幻想乡，月球也在办庆祝的宴席。正在这时，丰姬的某个发明物” 会动的巨大年糕 “, 向幻想乡飞去 ——
  • 解说⑤
    • 飞在幻想乡上空的 “会动的巨大年糕”，在以你为首的幻想乡居民们的协助与想吃年糕的斗争心下被粉碎了。
    • 飞在空中的 “会动的巨大年糕” 与镜饼很相似。镜饼是为了祈愿当年的丰收与庆祝新的开始 —— 作为节日食物是福气的源头。通过将它敲碎食用，能被其授予力量，是个吉祥的东西。

# 特性

  • 使用装备符卡时，受到来自妖怪的伤害下降 40%(1 回合)
  • 使用装备符卡时，阳攻将上升 3 等级 (1 回合)
