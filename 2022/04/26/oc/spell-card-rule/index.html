# Spell Card Rule

# 基本规则

  • 每张卡牌能力强度有五分之一来自「容器」，五分之四来自卡牌的使用者。
  • 使用者的能力为媒介，越强大卡牌可越超出预想。有下限，却无上限
  • 目前只有「镜子」拥有自我意识
  • 卡牌可以混合使用
  • 任何人都可以使用卡牌
  • 强大的使用者可以使卡牌的虚化作实
  • 使用 CD 也取决于使用者的能力强度
  • 同一能力的使用者，更强大的一方可以更好得使用卡牌
  • 过度使用可能会燃烧生命力。
  • [待补充]

# Spell Card

# 镜子 - Kakami

  • 复制非生物、非巨大质量的物品，可复制出虚 / 实体。(越强大的使用者，则可实现实体化)
    • ps：比如复制钥匙，可以实体化，且可多次复制
  • 复制生物、拥有意识的生物，则「镜子」本体实现实体化。
    • 复制生物所拥有的性格、记忆、能力、DNA 等独有特征，取决于使用者能力强度。
  • 虚化＜实体化＜生物复制＜独有特征复制＜完全复制
  • 「镜子」复制的所有生物默认与被复制舞一致，非镜像。
  • 默认复制所存在的事物，非存在、虚拟事物的实体化所需要的能力强度与复制生物强度一致。
  • 待补充

# 月神 - Tukigami

  • 治愈术，小到擦伤，大到致命伤。取决于使用者能力强度
  • 不可起死回生
  • 解毒术：世上必须存在解毒方法或存在制作解毒剂的秘方
  • 精神治愈：抹除精神伤的源头 (可以是记忆，也可以是情感，不建议使用)，是否能触发，取决于使用者能力强度。
  • 施法范围：取决使用者能力强度
  • 待补充

# 仓库 - Warehouse

  • 次元空间，一般储存各种各样的武器。
  • 一直是御守院馨使用，所以可能会有其他东西在吧。大概有餐具、家具、衣服日常用品，还可能有 “回忆”
  • 不可储存任何活物
  • 储存空间大小取决于使用者能力强度
  • 仓库只有一个空间。
  • 待补充

# 骑士 - Knight

  • 可召唤出骑士的卡牌
  • 数量取决于使用者能力强度
  • 出现地点可自定
  • 骑士强度取决于使用者能力强度，无法使用召唤者自身能力以外的能力。能力强度是召唤者的七分之一。体术除外
  • 所有骑士均无自我意识。
  • 待补充

# 创世 - Creation

  • 创世 → 创造
  • 原来的用途 创世 - 创造世界 / 世界式 已经无法再次使用。一次性 那么创世创造了什么呢？
  • 可创造存在的东西，或不存在的东西 取决于使用者能力强度
  • 无法制造拥有自我意识的东西。比如：动物、人
  • 制造数量取决于使用者能力强度
  • 创造越精致的东西可根据使用者的手艺 (物理)，或者取决于使用者能力强度
  • 待补充

# 星辰 - Star

  • 连接通道
  • 可联通人与人之间意识对话。
  • 充当网线
  • 可连接世界与世界的通道，也就是世界的备用钥匙。
  • 使用、连接范围取决于使用者能力强度。有人用出过天网
  • 待补充

# 黄昏 - Duskness

  • 使用者：蒲木泽夜
  • 因泽夜是黄昏的吸血鬼，只有在黄昏时能力会达到巅峰。所以黄昏可保持泽夜能力。
  • 因无视世界式的时间、天启的变化，所以本身能力是很逆天的
  • 真正的能力还未发现
  • 待补充

# 万物生 - Spring

  • 附有万物的能力
  • 春天的万物复苏；夏天的燥热；秋天的丰收、凋零；冬天的安眠。
  • 使用范围取决于使用者能力强度
  • 能力强度取决于使用者能力强度
  • 待补充

# 以太 - Ether

  • 火 + 水 + 木 + 金 + 土 + 风 + 雷
  • 一般无法与第三阶神相比 使用者能力高于第三阶神可以匹敌了吧
  • 适用范围取决于使用者能力强度
  • 使用方式取决于使用者
  • 待补充
