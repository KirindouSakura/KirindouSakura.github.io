# 第四阶神 - 人神

# 设定

  • 名前：御守院馨(ごもいんかおる)、Sweet
  • 性别：女
  • 诞生日：4.23
  • 年龄：900+ / 外表年龄：25 岁
  • 胸围：B+ 还有成长空间
  • 身高：174cm
  • 体重：56kg
  • 职业：神使 / 战斗女仆
  • 种族：人类 (意大利人)
  • 能力：能够熟练使用任何武器、幻术 (成为神使后，但很弱)
  • 武器：短刀

# 人际关系

  • 喜欢的人：鬼凛堂桜
  • 恋人：绯上灵琰
  • 信赖的人：蒲木泽夜
  • 上司：界、音
  • 仇视鬼神

# 简介

  • 曾经是人类，死的时候被招募成神使

    • 据说是因为以自身为交换实现了某个愿望
    • 生前是粉瞳，死后是银瞳。和灵琰灵魂交互后，右眼变成了蓝色

  • 生前好像也侍奉着谁。经常用的伪名：蝶岚

  • 象征物是蝴蝶🦋

  • 生前国家的主权者因和鬼神做交易，导致主和自己最后的落魄，所以很仇视鬼神。

  • 魔力很弱，一般以辅助为主。毕竟是人类嘛

  • 体术很强，任何武器都很擅长和现在的主打一架还不一定会输呢，魔法除外

  • 称鬼凛堂桜为：大小姐

  • 十分忠诚，无背叛经历

  • 虽然蒲木泽夜曾背叛过神，但依然信任着他

    • 经常捉弄蒲木泽夜
    • 在工作上经常刁难蒲木泽夜
    • 很喜欢揍蒲木泽夜 因为真的很欠打
    • 也可以是邪教 cp？！

  • 愿意为了别人学习任何事

  • 擅长手工、料理、家务、打架

  • 鬼凛堂桜的 第二个神使

  • SpellCardRule

# 执笔

  • “她将自己交付于神明，踹跹的蝶于永恒的契约上落下一吻。” ———— 弥光
  • “折翼的蝶坠入石蕊丛中，神社的风铃于檐角叮当作响。她们站在葱茏的山上，看时间从过去流得很长。” ———— 弥光
