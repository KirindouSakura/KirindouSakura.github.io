烟火中的一等星

# 烟火中的一等星 ☆☆☆☆☆

  • 阳攻 + 100
  • 阳防 + 80

# 解说

  • 解说①
    • 博丽灵梦
    • 没有重力的束缚，身为博丽巫女的她能够在空中自由地翱翔。
    • 在没有异变发生的闲暇夜晚，伫立于博丽神社前的鸟居上空俯瞰静谧的幻想乡，这份惬意只有身为大结界管理者的灵梦才能品尝到吧。
  • 解说②
    • 雾雨魔理沙
    • 身为人类，却在锲而不舍的努力下成为了一名优秀的魔法使。
    • 魔理沙的魔法不仅有着华丽的外表，火力也不容小髻。很难想象这是她仅凭一己之力收集材料、并加以实验而收获的成果。
  • 解说③
    • “烟花” 在香霖堂的引进下，早已被幻想乡的居民们用于宴会或是平日的嬉闹。
    • 明明直接放出弹幕更方便也更环保才是……!
  • 解说④
    • 不知是何处正在举办宴会，随着一声巨响，几道流星般炫目的光芒从地面升至幻想乡的夜空。随即，宛若花瓣的光粒便从空中悠悠坠落下来，给灵梦深红色的缎带染上几笔金色的同时，也倒映在她黑色的瞳孔中。
  • 解说⑤
    • 在吸引灵梦的视线这件事上，魔理沙是永远不会认输的。
    • 投掷出亲手制作的魔法道具后，不亚于烟花的璀璨在鸟居的周围迸裂绽放。魔法的冲击力下，魔理沙蓬松的金发也如同烟花一样 “砰” 地散开，仿佛就连她自己都要与身旁的光芒一同化作明星。

# 特性

  • 装备符卡的灵力回收效率上升 100%(1 回合)
  • 使用装备符卡时， 仇恨值将上升 3 等级 (3 回合)
  • 装备符卡使用时自身结界回复 1 枚

追忆之时

# 追忆之时 ☆☆☆☆☆

  • 速度 + 85
  • 阳攻 + 95

# 解说

  • 解说①
    • 八云紫
    • 落樱不再，烟雨蒙蒙。
    • 回首间，恍然如梦，终是幻影。
  • 解说②
    • 蝉鸣扰扰，银雪绵绵。
    • 玉阁春色清梦着，千年一隙间。
  • 解说③
    • 月影杳香，靛裳翩翻。
    • 华伞旋落境界现，何处幻乡何处眠。
  • 解说④
    • “那棵樱树” 即将开花之时，春色仿佛也溜进了梦中。
    • 悠悠转醒，千年时光像弹指一瞬，从隙间轻灵地溜走一般。
  • 解说⑤
    • 越是无法拥有之物，越是心存幻想。死亡是生者的幻想。
    • 这片幽深的幻想之境，会不会也只是隙间妖怪的悠然一梦呢。

# 特性

  • 装备符卡使用时受到来自人类的伤害下降 30%(2 回合)
  • 装备符卡的尖头弹的威力上升 20%(1 回合)
  • 装备符卡使用时阳防上升 1 等级 (2 回合)

# 绘礼 - 菊

一时兴起的晚宴

# 一时兴起的晚宴 ☆☆☆☆☆

  • 阳防 + 65
  • 阴攻 + 115

# 解说

  • 解说①
    • 蕾米莉亚・斯卡雷特
    • 500 岁的吸血鬼领主，红魔馆的主人。有着与年龄相配的气魄与能力，也不乏与外貌相衬的任性与孩子气。
    • “今晚来举办宴会吧～！”，既不是什么纪念日，也不是哪一位的生日，这般唐突的晚宴在红魔馆不过是家常便饭。
  • 解说②
    • 十六夜哄夜
    • 红魔馆的人类女仆长。作为蕾米莉亚无可挑剔的从者，只需一个响指便能将大家拜托的料理端上华丽的餐桌，虽然将时间停止后繁忙的身影只有她自己知道……
    • 即使是吹毛求疵的蕾米莉亚，对吠夜亲手制作的佳肴也誉不绝口。
  • 解说③
    • 芙兰朵露・斯卡雷特
    • 比蕾米莉亚年幼 5 岁的吸血鬼。有着破坏一切的力量。加上芙兰朵露起伏不定的情绪与好战的性格，她的存在仿佛是红魔馆的一颗定时炸弹。
    • 关在地下室的她很少参加宴会。红雾异变后，多亏了灵梦一行人与她的交流，这颗定时炸弹并没有引起什么太糟糕的事态。
    • 虽说是家常便饭的晚宴，还是希望不要发生混乱呢……
  • 解说④
    • 帕秋莉・诺蕾姬、小恶魔
    • 作为红魔馆的头脑，深居于地下大图书馆的魔女 —— 帕秋莉并没有精力将求知的时间用于图书馆的管理，因此，小恶魔似乎被赋予了图书管理员一职。
    • 虽然今天又让偷书的黑白老鼠逃走了，却也在追逐途中意外收获了红魔馆内难得一见的食材，不如交给哄夜来好好利用一下吧～
  • 解说⑤
    • 红美玲
    • 红魔馆的门卫，平日的寝食也与红魔馆的其他人一同进行。即使经常在偷懒时被自己的人类上司 —— 十六夜以夜教训，也从心底敬佩着哄夜的实力与她对蕾米的忠诚。
    • 有时也希望哄夜能够和自己一样尽情地 “偷懒” 一下呢。

# 特性

  • 使用时敌方单体阴防下降 3 等级 (2 回合)
  • 使用装备符卡时， 命中将上升 2 等级 (1 回合)
  • 装备符卡使用时受到来自吸血鬼的伤害下降 30%(3 回合)

墨染之樱

# 墨染之樱 ☆☆☆☆☆

  • 速度 + 95
  • 阴攻 + 85

# 解说

  • 解说①
    • 西行寺幽幽子
    • 墨染的樱花盛放之时，她是否回到了梦中的尘世？
    • 昨日雪如花，今日花如雪。
  • 解说②
    • 弘川埋骨，西行永眠。
    • 白玉墨染之时，蝶舞樱落。
    • 斯人已逝，唯庭阁相伴、扇月相和。
  • 解说③
    • 浮生若梦，愿银雪垂帘，春度安枕，悠然长醉，不复轮回永劫之思。
  • 解说④
    • 埋骨于弘川，永眠在不会开花的西行妖之下。
    • 当白玉楼被墨染樱花妆点之时，幽蝶纷飞、樱瓣零落。
    • 长久陪伴已逝少女的，是静默的庭阁与皓月。
  • 解说⑤
    • “古籍里记述的那人，到底是谁呢？”
    • 少女一心解开谜题，于是春雪如白银之帘一般，冰封幻想乡。
    • 聚集于此的春度，能让她暂且沉浸于梦境吗？可以的话，就让少女停留在她美好的幻梦中，永远脱离轮回之苦吧。

# 特性

  • 装备符卡使用时受到来自白玉楼住民的伤害下降 30%(2 回合)
  • 装备符卡使用时阴防上升 2 等级 (2 回合)
  • 装备符卡使用时受到来自体术的伤害下降 10%(2 回合)

# 绘礼 - 梅

灵异社团

# 灵异社团 ☆☆☆☆☆

  • 阳防 + 76
  • 阴防 + 74

# 解说

  • 解说①
    • 《在某个「膜」的某个「断面」上。好像有类似这样的话。》
    • 「最近怎么样，梅莉」
      • 「迟到了 1 分 20 秒」
      • 「是吗，赶上了呢」
      • 「完全没有赶上」
    • 莲子想要知道。姑且算是有痊愈的迹象了。那只是没有被他们的眼睛观测而已。莲子无论是过去还是现在。一直在观察梅莉异常而疯狂的能力。
    • 「和平时有点不同的活动呢」
      • 「和往常一样，到处多看看吧」「我并不讨厌到处转悠」
      • 「能满足公主的希望，荣幸之至」
    • 那么今天也。—— 开始秘封俱乐部吧。
  • 解说②
    • 宇佐见莲子
      • 灵异社团「秘封俱乐部」的成员。被人叫做「莲子」。
      • 在自己知道的人中，拥有最高的头脑。[M.H. 氏谈]
    • 玛艾露贝莉・赫恩
      • 灵异社团「秘封俱乐部」的成员。被人叫做「梅莉」。
      • 在自己知道的人中，最有神秘感。[R.U. 氏谈]
  • 解说③
    • 古代物理学迎来了终焉。电磁气力、弱力、强力，还有重力。已经被证明，所有的物理现象，只不过是万物的理论「超统一物理学」的一面而已。那是叫做实用向超弦理论，脑力世界……「膜宇宙」等概念，揭开我们世界尊颜的瞬间。
    • 作为现代人，那不过是一种「教诲」。物理学者们只是为神官，翻译从大宇宙中传来的神谕而已。然后他们也没有「单独的」了解到宇宙的原貌。所以至今还能看到统一外的「灵」。不同的人以不同的方式看到。灵能……「灵力之场」，就是指那个吧？
    • 那是根据每个观测者而变化的宇宙。不只是四个，把实际上无数存在的宇宙统一起来，这可能吗？莲子和梅莉一边聊一边想象着。
  • 解说④
    • 秘封俱乐部
      • 莲子和梅莉，在大学期间的社团活动。
    • 虽然是灵能者社团，但是像这种类似的灵异社团的降灵会或者除灵活动都并没有做过。
    • 所以在其他的灵能大学生眼里，她们是在霸占教室与预算，的的确确是不良社团的样子。
    • 那么，她们到底做些什么呢。
    • 其实，她们在通过结界和梦境「到异世界中冒险去了」。
    • 还有就是…… 把她们去异世界的体验做成同人志之类的？
  • 解说⑤
    • 医生的诊断是「病毒性儋妄」。
      • 赫恩家族传承的「寄宿于血脉的灵能」。莲子觉得无论哪个「都说的通」。但是这离「唯一的真理」还差的很远。
      • 让她察觉到这一点的就是玛艾露贝莉・赫恩。对于莲子来说的梅莉。常常是作为魔术师的梅莉。
    • 「很亲切的老师呢。帮我找到了病历卡，时间晚了，还特地用亲戚的住宿招待我们」
      • 「在我们蒙受关照的时候也一直非常亲切热情」
      • 清晨的信州。
      • 空气清新，心情凉爽。
    • 「那么，关于昨天晚上就很在意这个地图。」莲子这么说的时候。梅莉已经把裙子下摆撩起…… 正在跨过栅栏。哎呀哎呀，真是个了不起的公主啊。

# 特性

  • 装备符卡的追踪弹威力上升 35%(1 回合)
  • 装备符卡的镭射弹威力上升 35%(1 回合)
  • 使用时自身回避上升 2 等级 (3 回合)
