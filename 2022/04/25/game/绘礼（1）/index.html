# 绘礼 - 兰

吃食无双

# 吃食无双 ☆☆☆☆☆

  • 体力 + 600
  • 阴攻 + 60

# 解说

  • 解说①
    • 西行寺幽幽子
    • 白玉楼的主人，西行寺家的大小姐。对人类并没有任何怨念，拥有能足以统领所有幽灵的实力，因此被阎王任命去管理幽灵。要说交换条件是什么，她获得了永久居住在冥界的许可。
  • 解说②
    • 魂魄妖梦
    • 人类幽灵各占一半的少女。
    • 虽说是一半但却并不是混血，而是被称为 “半人半灵” 的种族。在位于冥界的白玉楼里，是作为服侍主人西行寺幽幽子的从者。虽说是剑术指导和兼职艺苑师，但剑道还尚未成熟。
    • 白玉楼的园艺师兼剑术指导。
    • “基本不存在无法斩断之物！” 一边这么说着一边向侵入白玉楼的可疑人员砍去。
  • 解说③
    • “所谓的 “秋”—— 是日本的第三个季节，同时也指日磨上的九，十，十一月。
    • 这个季节相对来说会变得比较凉爽，同时也是五谷杂粮和水果的收获期。可是在外面的世界，有着秋老虎这种说法，总感觉没法变得凉快
  • 解说④
    • 运动之秋 —— 听说蕾米莉亚要举办运动会的幽幽子，开始拼命进行特训了。
    • 因为好奇在做什么的妖梦靠近一看…… 发现是在吃吊起来的苹果！
  • 解说⑤
    • 比谁都更先一步解决苹果，比谁都更快把苹果吃掉。
    • 看到这副景象，不由得想到了一个好主意 “只要边运动边吃，肯定对身体有好处的吧！”
    • 在一段时间里，白玉楼拿出点心的时候，都会用咬苹果赛跑之类的形式……。

# 特性

  • 装备符卡使用时体力回复 40%
  • 装备符卡的体术的威力上升 45%(1 回合)

烟火中的一等星

# 烟火中的一等星 ☆☆☆☆☆

  • 阳攻 + 100
  • 阳防 + 80

# 解说

  • 解说①
    • 博丽灵梦
    • 没有重力的束缚，身为博丽巫女的她能够在空中自由地翱翔。
    • 在没有异变发生的闲暇夜晚，伫立于博丽神社前的鸟居上空俯瞰静谧的幻想乡，这份惬意只有身为大结界管理者的灵梦才能品尝到吧。
  • 解说②
    • 雾雨魔理沙
    • 身为人类，却在锲而不舍的努力下成为了一名优秀的魔法使。
    • 魔理沙的魔法不仅有着华丽的外表，火力也不容小髻。很难想象这是她仅凭一己之力收集材料、并加以实验而收获的成果。
  • 解说③
    • “烟花” 在香霖堂的引进下，早已被幻想乡的居民们用于宴会或是平日的嬉闹。
    • 明明直接放出弹幕更方便也更环保才是……!
  • 解说④
    • 不知是何处正在举办宴会，随着一声巨响，几道流星般炫目的光芒从地面升至幻想乡的夜空。随即，宛若花瓣的光粒便从空中悠悠坠落下来，给灵梦深红色的缎带染上几笔金色的同时，也倒映在她黑色的瞳孔中。
  • 解说⑤
    • 在吸引灵梦的视线这件事上，魔理沙是永远不会认输的。
    • 投掷出亲手制作的魔法道具后，不亚于烟花的璀璨在鸟居的周围迸裂绽放。魔法的冲击力下，魔理沙蓬松的金发也如同烟花一样 “砰” 地散开，仿佛就连她自己都要与身旁的光芒一同化作明星。

蓬莱的人之形

# 蓬莱的人之形 ☆☆☆☆☆

  • 阳攻 + 90
  • 阴防 + 90

# 解说

  • 解说①
    • 藤原妹红
      • 她是住在迷途竹林里，不老不死的少女。
      • 虽然原本只是普通人类，但因为吃了 “蓬莱之药” 而获得了蓬莱人不死的肉体。在那时，与和她不对付的辉夜同时拥有了不死的属性，作为宿敌时常进行（毫无意义的) 壮绝的厮杀。
  • 解说②
    • 妹红在成为不死之身以前就精通札术和阴阳道，再加上成为不死之身后，在超过千年的人生中获得了种种妖术与战斗经验。
    • 因此她的战斗力很高，如果是平均水平的妖怪能被她轻松击退。
  • 解说③
    • 对服用了蓬莱之药的人，人们称之为 “蓬莱人”。
    • 他们因药效拥有了不死者的特性，仅有灵魂成为了其存在的本体，哪怕肉体毁灭，也能自由生出新的肉体。
    • 因为是灵魂所以不灭，其肉体不会老化也不会生病，但因为身体并没有产生变化，所以会受伤，也会感到疼痛。
  • 解说④
    • 就算死去也会自熊熊燃烧的烈焰中重生，永生的传说中的不死鸟 “凤凰”, 又名为 “火鸟”。
    • 通过组合与生俱来的妖术与蓬莱人的特性，她实现了独特的战斗风格，作为描述它的词语，在各种意义上都没有比这更合适的吧。
  • 解说⑤
    • 身缠红莲之业火，将炽热燃烧的心意凭借本能释放出去。
    • 在这心中沸腾的是对往年宿敌的憎恶呢，还是能实际感受到生的瞬间带来的喜悦呢 —— 她自己是不是早就忘却了那种疑问呢。她露出的表情让人不禁这样想。

# 特性

  • 技巧式使用时自身阳防上升 3 等级 (3 回合)
  • 技巧式使用时自身会心命中上升 3 等级 (3 回合)
  • 使用时自身受到来自蓬莱人的伤害下降 15%(2 回合)

血之羁绊

# 血之羁绊 ☆☆☆☆☆

  • 阳攻 + 55
  • 阴攻 + 125

# 解说

  • 解说①
    • 蕾米莉亚・斯卡蕾特
    • 红魔馆的主人，被称为绯红恶魔。传说她是特佩斯一族的末裔。
  • 解说②
    • 芙兰朵露・斯卡蕾特
    • 恶魔之妹。既是吸血鬼，又是魔法少女。
    • 对蕾米莉亚姐姐十分仰慕，但姊妹关系仍有很多谜团。
  • 解说③
    • 所谓的 “弗拉德・采佩什”—— 德古拉伯爵的原型也是十五世纪罗马尼亚瓦拉几亚的领主。传闻中也常被称为刺穿公。采配什并不是姓氏，而是罗马尼亚语里 “吃穿之人” 的意思。
  • 解说④
    • 据说蕾米莉亚因为饭量小的缘故，会把那些吸不了的血都给弄撒，导致衣服上都被染得鲜红。看到了她那副样子的人都会害怕至极，会不会称她为红色恶魔呢......
  • 解说⑤
    • 整座红魔馆是从外面世界来的。
    • 其主人蕾米莉亚是与外表不同，生活了五百年以上的吸血鬼。但是就算是被吸血鬼的蕾米莉亚吸了血，只不过是会贫血罢了。
    • 不过，芙兰朵露只吃过吠夜和妖精女仆准备的食物，不知道如何袭击人类。因此如果被她吸的话或许会被吸得一干二净吧……。

# 特性

  • 装备符卡的火属性的威力上升 50%(1 回合)
  • 装备符卡使用时受到来自幻想乡的贤者的伤害下降 20%(2 回合)

一人舞台

# 一人舞台 ☆☆☆☆☆

  • 阳防 + 81
  • 阴攻 + 99

# 解说

  • 解说①
    • 秦心
      • 古旧的面具变成附丧神，名为面灵气的妖怪。看上去虽然是人类的样子，但本体是面具。
      • 另外，她的面具是能剧与狂言的原型猿乐之祖，秦河胜使用过的。
  • 解说②
    • 心持有的面具，各自的感情是固定的。戴上那面具感情就会变化，对附近的人也会有影响。
    • 福神与老翁的面具，是高兴的时候。猿猴的面具是困惑的时候，生气的时候是般若的面具，像这样子感情表现相当丰富。
    • 那也是理所当然的，她持有的面具有六十六副之多，每副面具都担当一种感情。
  • 解说③
    • “神楽”—— 是有着自室町时代以来约六百年以上的历史，不断表演传承下来的日本代表性的舞台艺术。其语言和曲调，将室町时代的样式保留到了现代。
  • 解说④
    • 神子本人曾说，心的面具是丰聪耳神子命令河胜制作的。
    • 另外，以前希望之面曾经遗失过，但恋说在地面的裂缝中捡到了，坊间传言也许是这引发了间歇泉骚动。
  • 解说⑤
    • 心被数次目击到在博丽神社等地表演能乐。据说那能帮助她维持精神安定。
    • 主要的观众是村子里的人类，对他们来说正式的能乐过于难懂了。于是猫藏协助她，以简单易懂为重点创作了新的剧目。这部本纳神乐演出，据说盛况空前。

# 特性

  • 回复式使用时， 阴攻将上升 3 等级 (3 回合)
  • 回复式使用时，回避将上升 3 等级 (3 回合)
  • 使用装备符卡时，受到来自地底住民的伤害下降 20%(2 回合)

# 一时兴起的晚宴 ☆☆☆☆☆

  • 阳防 + 65
  • 阴攻 + 115

# 解说

  • 解说①
    • 蕾米莉亚・斯卡雷特
    • 500 岁的吸血鬼领主，红魔馆的主人。有着与年龄相配的气魄与能力，也不乏与外貌相衬的任性与孩子气。
    • “今晚来举办宴会吧～！”，既不是什么纪念日，也不是哪一位的生日，这般唐突的晚宴在红魔馆不过是家常便饭。
  • 解说②
    • 十六夜咲夜
    • 红魔馆的人类女仆长。作为蕾米莉亚无可挑剔的从者，只需一个响指便能将大家拜托的料理端上华丽的餐桌，虽然将时间停止后繁忙的身影只有她自己知道……
    • 即使是吹毛求疵的蕾米莉亚，对咲夜亲手制作的佳肴也誉不绝口。
  • 解说③
    • 芙兰朵露・斯卡雷特
    • 比蕾米莉亚年幼 5 岁的吸血鬼。有着破坏一切的力量。加上芙兰朵露起伏不定的情绪与好战的性格，她的存在仿佛是红魔馆的一颗定时炸弹。
    • 关在地下室的她很少参加宴会。红雾异变后，多亏了灵梦一行人与她的交流，这颗定时炸弹并没有引起什么太糟糕的事态。
    • 虽说是家常便饭的晚宴，还是希望不要发生混乱呢……
  • 解说④
    • 帕秋莉・诺蕾姬、小恶魔
    • 作为红魔馆的头脑，深居于地下大图书馆的魔女 —— 帕秋莉并没有精力将求知的时间用于图书馆的管理，因此，小恶魔似乎被赋予了图书管理员一职。
    • 虽然今天又让偷书的黑白老鼠逃走了，却也在追逐途中意外收获了红魔馆内难得一见的食材，不如交给咲夜来好好利用一下吧～
  • 解说⑤
    • 红美玲
    • 红魔馆的门卫，平日的寝食也与红魔馆的其他人一同进行。即使经常在偷懒时被自己的人类上司 —— 十六夜咲夜教训，也从心底敬佩着咲夜的实力与她对蕾米的忠诚。
    • 有时也希望咲夜能够和自己一样尽情地 “偷懒” 一下呢。

月面战争

# 月面战争 ☆☆☆☆☆

  • 阳防 + 120
  • 阴防 + 60

# 解说

  • 解说①
    • 八云紫
    • 能够操纵神出鬼没的境界的妖怪。
    • 使用隙间能出现在幻想乡的任何地方，并委托灵梦去消除麻烦事。像这次的封结晶，第一个发现的人也正是紫。
    • 她微笑地说道:“因为我比谁都爱着幻想乡。”
  • 解说②
    • 「满月」是指 —— 当月球在太阳正对面位置时在地球看到的月相。如果没有发生月食的话，地球的一半都会受到太阳光的照射形成圆形的光圈。
    • 据说此时的亮度有金星最亮时刻的一千九百倍之多。另外，听说从前的月面上的花纹看起来像兔子，或者是其他什么东西，人类的眼睛真是有意思啊……。
  • 解说③
    • 数千年前的旧逸闻 —— 紫通过操纵虚实的境界，飞进了在湖面上的月亮倒影。攻入了与其相连的真正的月球。
    • 带了很多妖怪一起去，却被月之都得近代兵器击溃了。紫在那之后就没什么大动静了，但却让灵梦们制作住吉三神火箭，意图挑起第二次月面战争 ——
  • 解说④
    • 丰姬、依姬和铃仙二号收到了永琳写的信件后还将入侵者灵梦和紫一行人击退了。与此同时，幽幽子接受了紫的委托，和妖梦一同潜入了月之都。
    • 成功取得了千年老佳酿，让丰姬和依姬大吃了一惊。
  • 解说⑤
    • 如果，第二次月面战争并非不战而胜的话… 如果，第一次的时候带着妖怪她们，带着背负最强之名的鬼族和紫的挚友幽幽子又或者隐岐奈也好 —— 一定能变成一段佳话吧。
    • 如果，还会发动第三次月面战争… 那结果会是怎样呢？大概你也能想象到吧 ——

# 我所喜爱之物 ☆☆☆☆☆

  • 速度 + 94
  • 阳防 + 86

# 解说

  • 解说①
    • 茨木华扇
    • 别名茨华仙，是居住在妖怪之山的仙人。这位自称是作为人类的同伴的仙人，因帮助博丽巫女修行的事件而变得颇有名气。
    • 当然，作为那被强制修行的身子积累了不少东西不是吗。
    • 华扇喜欢动物也被动物们喜爱。之所以经常出现在博丽神社，是因为要来指导灵梦的修行。
  • 解说②
    • “华扇有着 “引导动物的能力”。因为这个，所以能够同动物进行沟通，也能读懂它们的心情。
    • 和动物们心灵相通，说到这种事情就会想到日本传说中的金太郎，可以说是相当经典的一个妖术吧”。
  • 解说③
    • 在华扇的仙界里有着不少的动物 ——【久米】年事已高的大鹫。曾负责帮华扇跑腿和支援。
    • 【竿打】年轻的大鹫，是久米的接班人。因为还不熟悉回家的路，跑腿回来总是会很晚，每次都让华扇很担心。
  • 解说④
    • 【黄帝】华扇能让龙石孵化出龙之子。
    • 【彭祖】作为茨华仙宅邸门卫的大老虎。
    • 【务光】以前在神社里大闹时被华年给捕获并饲养的雷兽。喜欢的东西是玉米。”
  • 解说⑤
    • “华扇的宅邸即在道场之中，时不时还能见到灵梦的身影。那个时候她一定是被强行带来的。让那样讨厌修行的灵梦勉强地做承诺的，到头来她的修行确确实实是为了自己不是吗。如果也想变强的你也是这么觉得的话，一只手拿着指南书一只手试着去敲敲道场的门会比较好吧。”

      • 她是生来的魔法使，据说活了百年以上。其中大多数时间都在读书中度过，心无旁骛地开发新魔法。
  • 解说②
    • 拥有 “操纵火＋水＋木＋金＋土＋日＋月程度的能力” 的帕秋莉，是很符合魔法使形象的魔法使，擅长属性魔法。
    • 所谓” 属性魔法 “—— 是藉助妖精与精灵的力量，就算是微小的力量也可能发挥出巨大效果的魔法。
  • 解说③
    • 帕秋莉擅长变化与运动的 “火”、静寂与净化的 “水”、生命与觉醒的 “木”、成熟与丰饶的 “金”、基础与不动的 “土”、主动与攻击的 “日”、被动与防御的 “月” 这七属性的魔法。
  • 解说④
    • 所谓” 魔导书 “—— 是融入了魔法的魔术道具。
    • 特别是在帕秋莉这里，她只要创造出了新的魔法，就会写进魔导书。也许就是因为这个，红魔馆地下的大图书馆里的书一直在增多。其中不光是记载了魔法使用方法，也许还记载了为使用魔法所需的关键秘密……
  • 解说⑤
    • 帕秋莉所持有的符卡中，有火水木金土符 “贤者之石”。这是将魔法的五个基本要素分别结晶化，将那些石头本身用魔法射出去。
    • 说到贤者之石，就是炼金术…… 但帕秋莉很轻松地就完成了。其成果，或许就是失言世界中使用的 “最后的贤者之石”。

  • 装备符卡的金属性的威力上升 35%(1 回合)
  • 使用装备符卡时，受到来自使魔的伤害下降 25%(3 回合)
