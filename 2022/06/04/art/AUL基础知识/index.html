# AUL 基础知识

AUL 全称：AviUtl。一款简单、简易的后期制作软件。
注意：此软件是 32 位系统，且没有 64 位。 当然你可以等有生之年大佬们的开发
AUL 开发者之一：最強のクズ（固有 bug 修复、速度提升、以及新功能添加）
其他：蛇色、鈴音、rigaya
链接：官网、x264 中文版本

# 版本

  • 日文 (原版)、英文、中文。
  • 每个版本的 aup 文件不共通。即 aup 无法在另一个版本里打开，请慎重选择使用版本。
  • 最新中文版本 1.70b、key:1kdu

名称按键其他
打开工程Ctrl+Onull
保存Ctrl+Snull
撤回Ctrl+Znull

复制预设Ctrl+C时间轴上的预设文件

取消选择范围Ctrl+A即全选导出范围
  • 常见问题
    1. 内存不够、内存溢出、乱码弹窗、报错。
    • 解答：安装 B 站视频 BV1v4411q7qd（av65924541）中的插件，保存工程关闭 AviUtl 重开（不一定完全解决）
    1. NG、图片、单帧输出、导出
    • 解答：菜单栏「编辑」→PNG 输出，或者勾选菜单栏「设置」→PNG 输出设置，打开 PNG 输出窗口
    1. 动图、GIF 导入不动
    • 用记事本打开根目录下的 exedit.ini 文件，找到「.gif = 图片文件」，将其改为「.gif = 视频文件」，保存并重启 AUL
    1. 透明通道、Alpha 通道、透明背景
    • 勾选视频文件属性框中的「读取 Alpha 通道」.ps：部分视频可能无法使用。
    1. 内存不够、内存溢出、乱码弹窗、报错
    • 安装 B 站视频 BV1v4411q7qd（av65924541）中的插件，保存工程关闭 AviUtl 重开（不一定完全解决）
    1. 扩展编辑、时间轴窗口跑到 屏幕外面 去了
    • 解答：拖动你的主窗口在屏幕四周转转，去把它给 吸附 出来
    1. 时间轴的虚线怎么显示
    • 解答：首先这个叫 BPM 网格。右键勾选 bpm 标尺，再右键网格设置，输入 BPM 和拍数即可显示 BPM 网格。
    1. 无法导出
    • 解答：x264 渲染器的视频压缩中，每个后缀有 ... 的东西全部都重新加载即可。

# 新建工程

  1. 右键 拓展编辑 ，新建工程。
名称简介其他
工程尺寸整个视频的分辨率， 宽x高推荐尺寸： 1920x1080 、 1080x720
帧速率整个视频的 帧率 ，帧速率是指每秒钟刷新的图片的 帧数 ，也可以理解为图形处理器每秒钟能够刷新几次。推荐帧率： 30fps
音频采样率音频采样率是指录音设备在单位时间内对模拟信号采样的多少推荐： 默认值：44100Hz
匹配导入的文件导入的视频原分辨率、帧率、音频采样率与为 最终视频的一致null
  1. 可直接将文件夹里的视频、图片拖入 AUL 中。跳出新建工程选项，即上述。
  2. 若 拓展编辑 无意间消失，则在菜单栏的 设置 中，找到 拓展编辑设置 即可。

# 打开工程

  1. AUL 保存的文件格式为：aup
  2. 使用中的文件格式为：lwi
  3. 打开方式：找到文件所在地 直接拖入 。或者，在菜单栏的文件中 打开文件 / 打开工程

# 拓展编辑

整个视频最重要、最基础的设置。贯穿整个过程。

  1. 主页面介绍
名称简介其他
时间轴按时间顺序串联的记录体系00:00:00:00 即：时、分、秒、毫秒
图层含有文字或图形等元素的胶片，一张张按顺序叠放在一起，组合起来形成页面的最终效果。图层越下，顺序越前。右键可修改图层名称。
主合成最终合成栏null
次合成图层合成的分组null

附加：

  • 默认黑屏是透明背景，若在次合成需要在主合成中达成 透明背景的效果 ，需要在次合成栏中右键，勾选 启用Alpha通道 。
  • 主合成、次合成可 单独导出 。
  1. 最右侧有个灰色竖线，贯穿全部图层，它是视频的终点。竖线在时间轴的位置，就是整个视频的时间长度。
    注意：每次准备导出的时候，请检查这竖线是不是在你需要的时间长度上。若不是可右键，选择 范围设置→最后物件位置的为最后一帧 。

# 标准属性

每个图片、视频文件均拥有的属性。（时间轴上显示为蓝色）

名称简介其他
X在工程上 左右 的位置负数为左，0 为正中间，正数为右（文本除外）
Y在工程上 上下 的位置负数为上，0 为正中间，正数为下（文本除外）
Z在工程上 前后 的位置负数为后，0 为正中间，正数为前（文本除外）
缩放率在工程上文件的 显示大小 ，默认值为：100%与 Z 轴不同，缩放率 没有位置变化
透明度在工程上文件的 透明效果 ，默认值为：0%数值越 大 ，透明度越 高 。
旋转在工程上文件的 角度 ，默认值为 0数值即为旋转的角度，默认正数为 顺时针 。
混合模式文件在其他文件中的混合效果。null

附加：浏览文件即在标准属性的状态下，可随意更改文件。

  • 图片仅可更改图片，视频仅可更改视频

# 附加效果

  • 右上角的 + ，可在文件基础上进行效果编写。
  • 详细见其他文章

# 镜头效果

建设摄像机效果： 时间轴右键→添加媒体物件→摄像机控制→摄像机控制

  • 摄像机效果在时间轴上显示为绿色。
    注意：摄像机的 目标层数 在摄像机效果的层数上，即为以摄像机为中心。 0 为图像的中心。
    建议此效果 实践 一遍会有更好的理解。
  • 摄像机文件必须要在 所有需要使用镜头效果文件 的最上一层。且文件必须开启左上角的 摄像机标志 。

# 次合成

在主合成上使用次合成的内容，需在时间轴上右键→添加媒体物件→次合成

  • 选择次合成即可选择目标
  • 次合成文件所拥有的长度则是次合成中时间轴的时间长度。
  • 播放速度可调整。
  • 基础属性及附加效果均可使用。

# 导出文件

  1. 菜单栏里 选择文件 ，找到 插件渲染 ，选择 x264 扩展渲染(GUI)Ex 。
  2. 保存类型选择 Mp4 格式。
  3. 文件大小可在视频压缩里，左边 质量(Quality) 调整。数值 越小 视频 越清晰 ，导出的 文件越大 。默认值为：23
    注意：数值 越大 ，主合成里的 效果可能会发生改变 。
