# 妖怪

# 设定

  • 名前：绯上灵琰、緋上(ひかみ)タマキ、 雪女
  • 性别：女
  • 诞生日：12.7
  • 胸围：B+
  • 年龄：1800+ / 外表年龄：24 岁
  • 身高：170cm
  • 体重：58kg
  • 职业：神使
  • 种族：妖怪
  • 能力：操纵一切有关液体的物质，能使其结冰。操纵蓝色的火焰。(成为神使后)
    • 结冰的寒气 AOE 范围最大只有半径 20 米。
    • 借助神主的力量范围可以扩大十几倍。(每次都会问神主，虽然没必要。)
  • 武器：无 能操纵冰使其变成武器

# 人际关系

  • 喜欢的人：御守院馨 交往中～
  • 主人：鬼凛堂桜
  • 上司：天羅
  • 认为很有趣的人：蒲木泽夜
  • 讨厌：人类 和天羅一样咧～说不定可以是知心友人

# 简介

  • 曾经是妖怪，死的时候被招募成神使。第二神使
  • 象征物是风信子 象征着重生
  • 对鬼凛堂桜的称呼是「iesu 大人」，即：イエス様。本人其实很讨厌这种称呼
  • 生前化身为 白唇鹿 和人类互动。
  • 本应该死去的她被御守院找到 (找药材路过)。
  • 被天羅收留，舍弃了以前的身份，改名换姓，以神使的身份活着。杀了很多人应堕入地狱，所以现在地狱使者一直在找这个空白人
  • 知道灵琰真正身份的人屈指可数。
  • 御守院馨是她的知心好友。
    • 耳坠是确定恋人身份的时候御守院送的。
    • 因为身份特殊原因，很少执行任务，所以御守院馨又制作了人形布偶送给她。
  • 身份是 雪鹿妖 。
  • 很在意自己的能力不足，平常一直在修炼能力。直至现在已经临近绝对零度，纯火的阶段
  • 打架的时候可以和御守院馨一起使用组合技。她们平常有空就会练习。
  • 因是不正规的妖怪，所以一部分能力寄于神主，即：神主可以使用部分她的能力。
  • 现性格有点病娇，对熟人很亲切。非常乐意帮助他人（人类除外）。朋友的朋友是人类，虽然嘴上说着不愿意，但还是会帮助的
  • 「回灵」是一种使自身魔力释放手段，可以达到平常魔力浓度的 3~5 倍。
    • 灵琰的「回灵」是使自身变回真正的「妖怪」姿态。
    • 魔力大幅度提升，以自身为中心的寒气大范围扩散。
    • 打人更暴力了（？

# 详细

生前经历

1. 本是雪女的她一直守护着那座雪山和人类也挺友好，但因为气候变化，以及有心人对雪山的破坏，雪女的造谣。
2. 雪山的气候十分恶劣，导致雪崩。
3. 村庄大部分被雪掩埋，存活的人们就对所谓雪女的妖怪十分厌恶，决定把她除掉。
4. 无论雪女再怎么解释，人类依然不信，外来的人类因为了开发自己的地盘，所以需要雪山上的药材等可开发材料，加入了除掉雪女这行动。认为守护者不在了，就没有危害了。再次引发了雪崩，人类在山脚放了火。
5. 雪女被人类捆锁着、身上贴满了符咒，使用不了能力，被烧得面目全非。
6. 雪女的愤怒使红色的火焰变成愤怒的蓝色火焰，蔓延大部分雪山，在雪山上的人类大部分都被烧死了
7. 本应该死去的雪女被御守院找到 (找药材路过)，御守院觉得这经历和天羅很像，说不定能救她。天羅也被人类背叛过，决定救，但是杀了很多人，而且已经是已死的彼岸之人，不能复活了 (神规，复活的人必须上报)，所以只能变成神使
8 舍弃了雪女的身份，取名为绯上灵琰。如果用雪女的话，肯定会被查。
9. 之后灵琰一直被天羅照顾。称天羅为：神主様

# 人形布偶

  • 御守院馨亲手制作的布偶。里面充满了对对方的爱意。
  • 灵琰一直带在身边。谁也不可以摸。
  • 布偶身上带有御守院馨的香气，令人安心。
  • 执行任务的时候，会藏起来，避免损失。(虽然御守院馨说不见了她还会再做一个，但是灵琰却一直珍惜。)
