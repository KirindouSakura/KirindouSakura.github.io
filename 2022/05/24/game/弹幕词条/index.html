# 基本知识

弹幕是任何角色都持有的攻击手段。每位角色都拥有四种弹幕。

# 普通弹幕

  • 扩散：可以任意使用。通常威力较低，hit 数高，且灵力回收效率较高。
  • 集中：可以任意使用。通常威力较高（约为同范围扩散弹幕的 1.67 倍），hit 数低，灵力回收效率较低。

# 符卡

  • 普卡：每个角色拥有两种（每种各两张）符卡。每张符卡一场战斗只能使用一次，威力强大（约为同范围集中弹幕的 2倍 ）。符卡大多带有弹幕词条和发动前 / 发动后特效。
  • LastWord（LW）：终极符卡，每个角色只拥有一张，且一旦使用会令所有普卡全部变得无法使用，威力极大（约为同范围普卡的 2.5倍 ）。LW 大多带有弹幕词条和发动前 / 发动后特效。
    *备注：部分角色的普卡和LW是纯辅助符卡，仅有发动前/发动后特效，没有攻击性。
  • 弹幕有两种 攻击范围 ：全体和单体。原则来说，同类别的单体弹幕对个体造成的期待伤害（即 CP，详见下方的 “CP 计算式” 段落）是全体弹幕的 1.5倍 ，且每一 hit 能回收的灵力高于全体弹幕。

# 弹幕等级

  • 每个弹幕均有自己的等级，以 Lv.X 的形式表示。
  • 普通弹幕 加成从 Lv1 开始出现。其等级与当前灵力值的个位数相等。
  • 符卡 加成从 Lv2 开始出现。其等级与角色星级（觉醒等级）相等。
  • 总体来说，扩散的成长率最低，符卡中等，集中最高。
  • 等级最大时各弹幕的平均威力倍率：扩散约 122% ，集中约 130% ，符卡约 128% 。

# 弹幕构成

  • 每个弹幕均由 6 个 Bullet （段落）构成。通常情况下，弹幕使用时只会发射其第一个段落，玩家可以通过消耗 1~3 灵力进行 Boost（爆发）来激活后续的弹幕。详见下方的 “ Boost 和弹幕模板” 段落。
  • 部分符卡拥有 发动前特效和/或发动后特效 ，在该符卡发动前和发动完毕后触发。
  • 此外， 弹幕的构成与效果的发动等有关 。
  • 本游戏中，所有弹幕的动画以 0P 段落为基础构建。随后，其它段落的攻击会从前往后依序插入其中，构成完整的 6 段弹幕。
    弹幕附带的所有词条将在本段落的弹幕被激发时发动。

# 段落的属性

  • 每个弹幕段落均拥有以下属性：
    • 名称：段落的名字。
    • 阴阳：段落的阴阳气属性。段落标签最右侧的颜色（战斗中则是边框）表示该弹幕是阴气还是阳气，白色为阳气，黑色为阴气。阳气弹幕使用角色的阳攻和目标的阳防属性计算伤害，阴气弹幕使用阴攻和阴防。
    • 属性：段落的攻击属性。段落标签的颜色（战斗中是边框和文字）表示其攻击属性。颜色和属性的对应关系见下方表格。
    • 弹种：该段落使用的弹种。部分绘札对特定弹种有威力加成。
    • 攻击次数：标注在段落标签的右下角，表示该段落的总攻击次数。
    • 威力：该段落每一次攻击的伤害倍率。在计算时会乘以角色的攻击力。
    • 命中：该段落每一次攻击命中目标的几率。
    • 暴击：该段落每一次攻击造成暴击（造成 200% 伤害）的几率。
    • 词条：该段落附带的词条。

# 绘礼

  • 普卡和 LW 两种弹幕可以装配绘札。绘札可以提供两种效果：
  • 常驻属性加成 ：每张绘札为角色提供两项基础属性的加成。
  • 特效 ：装配绘札的符卡被使用时，会触发绘札的特效，给角色暂时附加指定的能力变化和瞬间效果。

# 弹幕词条

  • 弹幕词条附加在弹幕的段落上，为其添加额外的效果。
    注意：弹幕词条仅影响其所在的段落，不会对未带有词条的段落造成影响。
  • 备注：红字内容暂未实装，仅供参考
词条属性附加
必中该段落无视命中率计算，强制命中目标null
特攻该段落攻击特定种族的目标时，必定造成暴击null
贯通该段落无视目标因能力变化导致的 阳防阴防 变动null
镜面该段落以对手相反属性的防御计算伤害（例：阳攻对阴防）
硬质该段落将自身阴防或阳防的一部分附加到最终攻击（加成比例由各弹幕分别决定）
斩裂该段落将自身速力的一部分附加到最终攻击（加成比例由各弹幕分别决定）
速射该段落无视目标能力和特性造成的伤害减免null
爆破敌人有防御 BUFF 时，该段落容易暴击null
弹性敌人有攻击 BUFF 时，该段落容易暴击null
精密敌人有速力 BUFF 时，该段落容易暴击null
反动该段落 被发动时 消耗自身一定量体力，不会致命（体力消耗量由各弹幕独自设定）
吸收该段落 每次命中时 自身恢复一定比例体力（体力回复量由各弹幕独自设定，显示比例为全弹命中效果）
能力变化该段落 被发动时一定几率 附加同名的能力变化（例如，注目上升弹会赋予角色指定层数的注目增加）
烧夷该段落命中时一定几率施加一层「燃烧」结界异常null
冷却该段落命中时一定几率施加一层「冻结」结界异常null
荷电该段落命中时一定几率施加一层「带电」结界异常null
喷毒该段落命中时一定几率施加一层「毒雾」结界异常null
烟幕该段落命中时一定几率施加一层「黑暗」结界异常null
粘着该段落命中时一定几率对目标施加瞬间效果「延迟」null
冲击该段落命中时一定几率对目标施加瞬间效果「晕眩」null
烧却该段落命中时破坏目标所有处于「燃烧」状态的结界（触发时，属性克制造成的结界破坏不会发生）
冰解该段落命中时破坏目标所有处于「冻结」状态的结界（触发时，属性克制造成的结界破坏不会发生）
闪光该段落命中时破坏目标所有处于「黑暗」状态的结界（触发时，属性克制造成的结界破坏不会发生）
过毒该段落命中时破坏目标所有处于「毒雾」状态的结界（触发时，属性克制造成的结界破坏不会发生）
放电该段落命中时破坏目标所有处于「带电」状态的结界（触发时，属性克制造成的结界破坏不会发生）
