# 绘礼 - 兰

狗狗点单

# 狗狗点单 ☆☆☆☆

  • 体力 + 260
  • 阳攻 + 58

# 解说

  • 解说①
    • 犬走椛
    • 她是守护天狗住处的看门狗，不，是看门妖。那样的她今晚坐在 “古老欲望” 的吧台边一个人静静地喝着玻璃杯中的饮料，难得地陷入了沉思……
  • 解说②
    • 所谓 “将棋”—— 是在画着 9×9 格子的棋盘上，两名棋手各摆二十枚棋子进行的棋盘游戏。棋手交替从移动・打入・吃子・升变中选择一项，先吃掉对方 “王将” 棋子即为胜利。
    • 在通常的对局之外，还存在使用相同道具进行的其他游戏，以及地方上独自的规则。说起来，这家店的吧台旁也悄悄地放着将棋盘……
  • 解说③
    • 幻想乡中也存在将棋。不过，前述的概要只是人类使用。与主要在白狼天狗及河童之间流行的将棋完全不同。天狗们觉得人类用的规则对局时间太短了，便独自追加了棋子和规则。在妖怪们之间，将棋称为 “天狗大将棋”。在那之中彬也属于在积极扩充规则的一方，她对将棋的热情可见一斑。
  • 解说④
    • 「青娥娘娘的」淑女的社交场
    • 自从开始观察椛已经过了多久呢。她似乎终于想起来手中的玻璃杯已经空了，又叫了一杯限定饮料。接到点单后，青娥马上用熟练的手法摇起调酒器。她将做好的限定饮料注入玻璃杯，在吧台桌上滑过去。华丽又潇洒的演出…… 但是，椛对此毫不理会。
  • 解说⑤
    • 突然，椛轻声说了些什么。
    • “一边用‘振飞车’逼近敌阵，一边用狗符「狂犬断噬」制造空当 —— 不，要是被用弹幕压制会很难办…… 那么用‘穴熊围’坚守…… 千脆，用山窝「遭放逐者的约定之地」——” 虽然完全不知道她在念叨什么，但从她认真的表情来看，一定是对她很重要的事情吧。

  • 装备符卡的木属性威力上升 30%(1 回合)

吸血鬼的传统礼服

# 吸血鬼的传统礼服 ☆☆☆☆

  • 速度 + 65
  • 阳攻 + 45

# 解说

  • 解说①
    • 十六夜咲夜
    • 在红魔馆里工作的女仆长。能用保持冷静的样子去实现大小姐蕾米莉亚的各种任性要求的完美女仆。这次的要求是，听说由美铃制作出的中华料理十分美味，决定用 “即便是我也能够做到那种地步！” 来露一手的咲夜，开始了修行之旅。
    • 去人类的村落买东西的时候，感觉周边有些骚动。好像是在准备祭典的样子…… 然后想起了今天是举行某个仪式的日子。
  • 解说②
    • 蕾米莉亚・斯卡雷特
    • 永远的红色幼月，红魔馆的主人。
    • 在咲夜出去买东西的时候，邀请原本是打算邀请帕秋莉举办茶会的，可突然注意到了某件事。大事不好，不得不赶紧召开紧急会议 ——!
  • 解说③
    • 十月三十一日 —— 根据某一说法，这一天将会举行某个祈祷的节日的前夜祭。祝福秋收，迎接祖先的灵魂……・也就是和盂兰盆节差不多的节日。为了赶走同时出现的恶灵，人们会穿上奇装异服一边热闹举行祭典一边咏唱某种咒文。
  • 解说④
    • [蕾米莉亚 & 露米娅的] 捣蛋 or 弹幕
    • 蕾米莉亚突然召集了红魔馆的所有人，开始了紧急会议。要问理由的话…… 在食料仓库里的点心材料全部不见了！为了抓住偷走红魔馆东西的无礼之徒，不得不伪装起来！”
  • 解说⑤
    • 身为吸血鬼的蕾米莉亚，在寿命方面十分强大。并不会和人类一样害怕寿命论，就连生病受伤什么的也不会构成威胁。这次，由于一边寻找犯人的一边在幻想乡里东奔西跑，多多少少对于人类这方面的理解更加深化了…… 也说不定。

  • 装备符卡使用时受到来自姐姐的伤害下降 10%(2 回合)

# 绘礼 - 梅

来一个怎么样？

# 来一个怎么样？ ☆☆☆☆

  • 阳防 + 70
  • 阴防 + 40

# 解说

  • 解说①
    • 秦心
    • 她看上去是虚张声势的徒劳之人。
    • 在举办的 “报春杯・经典赛” 中，为了让大家精力充沛而开始销售便当。一边吃着 “孙子心便当”，一边精神地为选手加油怎样？
  • 解说②
    • 键山雏
    • 她是除厄的雏领队。
    • 为了精力充沛地为选手加油，向观众分发饮料。被厄运缠身无精打采的人喝了也能立刻精神起来的精品。其名为 —— 河童牌万能能量饮料尻子・黄瓜味！
    • 味道是受到了河童们认可的。
  • 解说③
    • “啦啦队队长”—— 是在体育赛事等的观众席上指挥助威的领袖。另外，啦啦队也有鼓励・鼓舞的意义在里面。不是指一个人，而是指由复数人构成的组织或队伍。
  • 解说④
    • 雏的助威・传说
    • 寒冷的季节过去，幻想乡开始暖和起来。莉莉白差不多该来通知「春天到啦～！」了…・… 本以为如此，却不管过了多久都没来通知。这是异变的前兆吗！？
  • 解说⑤
    • 不情愿地通告完春天的造访后就回去了。往常比任何人都对春天到来感到欢喜，精力充沛的她无影无踪了！
    • 到底发生了什么呢…… 不，不只是她。整个幻想乡都无精打采了！？原因是 —— 大型厄运一号『尤加林』!
    • 为了守护幻想乡的精神气，站出来吧 —— 五人伴奏 “报春五人组”!

# 特性

  • 使用时敌方单体阳攻下降 2 等级 (2 回合)
  • 装备符卡的体术威力上升 20%(1 回合)

# 绘礼 - 竹

秘密爱好

# 秘密爱好 ☆☆☆☆

  • 阳攻 + 56
  • 阴攻 + 54

# 解说

  • 解说①
    • 纯狐
    • 被怨恨纯化的灵化作了神灵。
    • 她的身上谜团重重，甚至她本人都一度对探究自身的情报失去了兴趣。而就在今天，她为自己增添了一条新的情报。
  • 解说②
    • “小号” 是一种在金属管中吹气、同时结合嘴唇的震动发声的金属管乐器，通过活塞阀改变调音管的长度从而控制音高。
    • 由幻想乡中梅露兰的地位就可以知道，在管弦乐团中小号常常担当独奏，可以说是乐团的核心之一。
  • 解说③
    • 【演奏吧！普利兹姆进行曲！】
    • 没有观众也没有听者的仙界中，纯狐独自吹奏着小号。就在数刻之前，在幻想乡四处游荡寻找新鲜事的纯狐，听说了某个演奏进行曲的乐团。她一定是想露上一手，让所有人大吃一惊吧。
  • 解说④
    • 无论是怎样强大的乐手，无论是怎样简单的乐器，演奏的水平也不是一朝一夕能够练就的。然而纯狐只是全神贯注地持续着吹奏。
    • 连仙界的法则都能够颠覆的罕见强者赫卡提亚其实已经观察了她好一阵子，但看她专注的劲头，想来是完全没有发现吧。
  • 解说⑤
    • 克劳恩皮丝被主人赫卡提亚千叮万嘱千万不要去打扰纯狐，但她不小心说漏嘴这种事大概早该料想到才对。
    • 看看望眼欲穿等着纯狐归来的光之三妖精，之前发生了什么就可想而知了。
    • 而且，在光之三妖精的身后还跟着几位音乐的行家呢。

# 特性

  • 使用时自身命中上升 2 等级 (3 回合)
  • 使用时自身受到来自神明的伤害下降 30%(3 回合)

恶魔大小姐

# 恶魔大小姐 ☆☆☆☆

  • 阳攻 + 55
  • 阴攻 + 55

# 解说

  • 解说①
    • 蕾米莉亚・斯卡蕾特
    • 红魔馆的主人，芙兰朵露的姐姐。
    • 因为是吸血” 鬼 “，所以怕撒豆子…… 本该是这样，但她说现在完全不值一提。其中理由，说是因为吃了豆子变强了。
  • 解说②
    • 芙兰朵露・斯卡蕾特
    • 蕾米莉亚的妹妹，家里蹲吸血鬼。
    • 她据说也和蕾米莉亚一样怕豆子，但飞过来的豆子都 “握住之后炸掉了！”，真相无法得知……
  • 解说③
    • 关于 “节分”—— 立春・立夏・立秋・立冬的前一天叫做节分 —— 也就是分开季节的意思。另外，过去立春是一年之始，它的前一天既是节分又是除夕。面对即将到来的新年，作为除厄消灾的仪式撒豆子，这仪式持续到了今天。
  • 解说④
    • 鬼与女仆与巧克力
    • 蕾米莉亚听说博丽神社在撒豆子，为了证明自己不再怕豆子而来。她不光是纳豆，还吃了咲夜准备的 “神秘豆子”，已经不必再害怕豆子了……… 但愿如此。
  • 解说⑤
    • 在博丽神社的撒豆子仪式上，被豆子痛打的萃香与勇仪。偶然来玩的华扇，代替阿吽扮演鬼。
    • 就算是撒豆子，也看不得动物被虐待！就算那是狛犬…… 说起来也确实是华扇的风格。

  • 装备符卡的通常弹的威力上升 15%(1 回合)
  • 装备符卡的镭射弹的威力上升 15%(1 回合)

纯白的圣夜

# 纯白的圣夜 ☆☆☆☆

  • 体力 + 300
  • 阴攻 + 30

# 解说

  • 解说①
    • 铃仙・优县华院・因幡
    • 住在用永远亭的永琳之弟子。
    • 地上的妖兽与妖怪兔有几点性质上的巨大差异。她毫无兽性，人们见过她礼节端正地向人类卖药。
  • 解说②
    • “操纵狂气程度的能力” 中的” 狂气 “—— 是让精神疯狂，操作寄宿于事物中的波动。波长变长的话，就会突然变得悠闲失去干劲，无法行动。反之变短的话，就会变得疯狂情绪不安定起来，会感情用事无法与人对话。
  • 解说③
    • 铃仙之所以会来地上，是为了逃避在月球发生的战争。如果地上之人没侵略月球的话，铃仙就不会逃亡到幻想乡也不会遇到永琳，或许会和绵月姐妹一同守护月之都。
  • 解说④
    • [辞旧迎新] 弹幕突击者们
    • 从永琳那里收到 “因为今年一年都做了个好孩子” 的夸奖和跟” 送礼的怪物 “故事中登场人物同样的衣服。那衣服十分温暖，正适合寒冷的冬天。因为包含着平日无法开口诉说的感谢之情，所以会感到更加温暖。
  • 解说⑤
    • 协助永琳可说是轻率的 “在年内给幻想乡全部好孩子送礼物” 这个计划的铃仙她们。制定高远目标，学习用不放弃努力下去的重要性。相信着最后可以大家一同欢笑 ——

# 特性

  • 使用装备符卡时， 会心命中将上升 2 等级 (2 回合)
  • 装备符卡的能量弹的威力上升 15%(1 回合)
