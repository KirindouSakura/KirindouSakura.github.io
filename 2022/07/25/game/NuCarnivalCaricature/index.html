# 新世界狂欢（NuCarnival）

  • 来到卡莱因大陆的伊得，得知自己必须代为履行休伊的任务，学习调节身上的魔力，并与眷属们进行魔力的「深度交流」──唯有解除霓虹宝石的封印、使其恢复原有的力量，伊得才有可能回到自己的世界……
  • 几百年前，大魔法师休伊为了平息卡莱因大陆上混乱的元素精灵之力，在五处祭坛设下封印。二十多年前，休伊突然失去踪影，导致平衡开始倾斜，天灾爆发、死地蔓延。休伊的使魔们透过秘术，召唤出了来自异世界的主角：伊得。

# 人物短漫

短漫展开

# 伊得 - Eiden

  • エイト様のことについてもっとご紹介したいと思います。
    お恥ずかしながら、わたくしその「グッズ」についてはあまり詳しくないのですが、どうやらみんなをいい気持ちにしてくれるもののようです。
  • しかしこの「ディル〇」というのは一体何なのでしょうか……？

# 艾斯特 & 墨菲 - Aster&Morvay

  • エイト様はどうやら最近寝つきが悪く、そのせいで元気がなく、筋肉痛になったりしているようです。どうやらふたりの使い魔のせいだとか……
  • それでは、わたくしエイト様のためにスープを作らなければなりませんので、これにて失礼いたします。

# 八云 - Yakumo

  • 八雲様はお料理が大好きで、いつも厨房で新しいレシピを試しておられます。
    先日、八雲様がエイト様に心を込めた新メニューをお出しになられた時……
  • ひょんなことから、八雲様が暴走していしまい、あやうく大惨事に！

# 艾德蒙特 - Edmond

  • 貴族出身で高い教養を持つエドモンド様は、普段あまり自分の気持ちに素直になれないでいます。
    普段エイト様に自分から近づいていくときも、毎回無理のある言い訳をしております。
    素直になれないツンデレとはまさにこのことですね。それもまたエドモンド様の魅力でございます！

# 奥利文 - Olivine

  • ご主人様は第一回の漫画の内容を覚えておりますか？
    新世界にやってきたエイト様が、なんとあの「オモチャ」を完成させたそうなのです！
  • オリビン様はどうやら少し… 期待している？ようなご様子ですね。

# 昆西 - Quincy

  • 森に住むケシー様と黄昏の星シュールベクス・ルシヴァラ (中略) トパのお話です。
    ケシー様の頼れるパートナーであるトパ。おふたりは普段どのように過ごされているのでしょう？
  • ちょっとだけ覗いてみましょう！

# 玖夜 - Kuya

  • 世界各地の珍しい品々をコレクションする玖夜さま。
    家の中には様々な素敵な「服」以外にも、ユニークなデザインの「道具」があるようで……。
  • エイトさまはコレクションを拝見したことで、新しいグッズのアイディアが浮かんだのではないでしょうか？

# 可尔 - Garu

  • 穏やかな性格のガルさまと比べて、一匹狼のカイルさまは気性が荒く、エイトさまの命令にも従おうとしません。
  • とはいえ…… エイトさまはどうやらカイルさまを従わせる上手い方法を見つけたようです。

# 布儡 - Blade

  • 彫刻の本を読み終えたばかりのブレイド様は、すぐさま三日三晩かけて、「ダーリン」へのプレゼントを手作りしたようです。
  • 目を輝かせながら、褒められるのを期待しているブレイド様に、果たしてエイト様はどのような言葉をかけるのでしょう？

# 啖天 - Dante

  • エンテン様は日ごろから努力を惜しまず、自分にも他人にも非常に厳しいお方です。天気がいい時は必ず訓練を行うと決めているエンテン様は、納涼中のエイト様を連れていかれたようです。でもおふたりの「訓練」とは一体……？
