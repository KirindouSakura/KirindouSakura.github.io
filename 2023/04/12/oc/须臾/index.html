# 掌权者 / 统治者

# 设定

  • 名前：须臾
  • 年龄：6000+
  • 胸围：A+
  • 身高：168cm/146cm
  • 体重：58kg/43kg
  • 种族：神阶统治者
  • 能力：拥有瞬间的能力
    • 拥有虚无的能力

# 人际关系

  • 亲友：伊邪那美、伊甸、天照
  • 挚友：枯荣
  • 敌人：不认同她的任何人。
  • 讨厌：拒绝艾墓尔欣邀请的神

# 简介

  • 拥有可以统治神阶的能力，但只有伊邪那美、伊甸知道。挚友枯荣没兴趣知道，所以从来没有问过。
    • 其他人只知道须臾拥有第二能力来统治神阶，并不知道具体的能力。
    • 除了本拥有的能力，也拥有整治好神阶的自身能力。
  • 病娇，死鱼眼。
  • 平时本来不会露脸，只有第一阶神和部分第二阶神的人知道须臾的样貌。Sakao 事件后正式出场。
  • 为了维持神阶的平衡和统治，可以不择手段。同族也能判死刑。
    • 曾经神的大战，须臾站在修改神式一派，殺了很多违抗这个想法的神。
    • 圣蕾拉计划的主谋之一。
    • 九刑伪物的主策划。
    • 「祈愿容器」计划的也是她用第二能力篡改的。
  • 讨厌无聊的事，经常去找乐子。喜欢一些无法预测的事情。
