# 人类

# 设定

  • 名前：白蕾拉
  • 性别：女
  • 外表年龄：24 岁
  • 诞生日：2.5
  • 种族：人类
    • 天香语山的巫女 / 人神
  • 胸围：B+

# 人际关系

  • 儿子、女儿：界、祈 / 桜
  • 丈夫：音
  • 孙女：七泉夜澪

# 简介

  • 原来是人类。后来成为人神。（在当时，人神也是神）
    • 是天香语山神的巫女
    • 天香语山的巫女历代都守护着这座山
    • 白蕾拉受当地人们的尊敬
  • 白蕾拉和音相爱。却因为寿命白蕾拉自愿成为人神，从而也成为天香语山的其中一位神。
  • 和音的第一个孩子是「界」。
    • 白蕾拉和音的恋爱超越了神和人之间的境界。
    • 「界」的能力则便是 「境界」 现在对外声称自己是式姐的界
  • 那个时候神没有属于自己的世界式，也就是说会因为信仰不足而消失。白蕾拉是天香语山神的巫女也是人神，信仰皆是天香语山神的，白蕾拉成为人神后只能待在山里，无法离开，与世隔绝。
    • 白蕾拉对外声称是自己身体原因，无法走太远的路。（人们不知道白蕾拉已经是人神了）
    • 白蕾拉的父母也以为是身体原因，经常请大夫们过来检查，但并没有什么异常
    • 因人神无法衰老，后来白蕾拉和父母坦白了自己的身份，父母表示不理解，音是别的神，和别家的神恋爱会遭天谴。白蕾拉不后悔，她是真心爱着音。但这种丑闻依然传出去了，人们都说白蕾拉不圣洁，信仰着别家的神。
    • 信仰逐渐消失。白蕾拉也因为信仰不足的原因死亡
  • 守护天香语山的巫女死亡，天香语山也一年不如一年，植物不规则死亡和生长，气候无规则变化，逐渐恶劣。真正的山神生气了，试图赶走山里的人们。
  • 虽然人类忏悔，但山神依然不原谅他们，把山封锁了，白蕾拉的家 / 山神的神社也永远藏匿在山的深处。
    • 音也不愿意接受白蕾拉的死亡
    • 策划了圣蕾拉计划。

# 圣蕾拉计划

机密，且勿随便浏览

此计划请勿随意传播
1、圣蕾拉计划实则是复活神。神是因为人类而生，以人类的信仰而亡。找回逝去的灵魂。
2、拥有绝望的地方，始终充满希望。所以以绝望为源，创造实现任何愿望的「祈愿容器」，超越世界式的力量。
3、音做出了装绝望和希望的容器。但这个计划泄露出去了，因为天香语山神是信仰很多的神，却也因为人类而消逝，众神为自己的信仰而害怕，人是会变的，能够预言未来的神也说过，众神将葬身于人界。人界将是八百万神明的坟墓。
4、甚多神威胁音，创造能改变世界式的祈愿容器，而不是单单复活神这个自私的愿望。音和界和众神打架，但没打赢。
5、这个时期神分为了两派。一派希望自己能因人类而亡，一派希望神能与人断绝信仰关系。打了 200 多年的战争。
6、神互相残杀几乎失去了 60% 的神。
7、容器是音为了白蕾拉做出来的，也用了白蕾拉送他的信物 —— 神楽铃。音不愿意把容器交出去，擅自修改了计划，让容器拥有自我意识，不成为神的工具。但计划赶不上变化，出错了。
8、容器确实拥有了自我意识，也拥有了实体，但这个自我意识很薄弱，很随机。如人偶一般。缺的是魂、灵。失去白蕾拉的信物为代价，让容器拥有了魂。祈便诞生了。
9、白蕾拉在这个世上除了被封锁的天香语山里的神社，再无任何证明过她存在的东西。任何形式的存在都具有保质期
10、而因为圣蕾拉计划，导致神的战争，失去了几乎 60% 的神，所以圣蕾拉计划一直是神们的禁词，白蕾拉也是禁词，也被禁止回到圣蕾拉计划的那几百年。界也被迫无奈对外声称自己是世界的界。真相永远被埋葬。
11、祈的自我意识很随机，但这个随机性唤醒了天羅。天羅的能力可以调节祈的不完整性。
