# 伪神

  • 无神髓的神
  • 非纯种的神，可能是其他种族因某种渠道成为的 灵
  • 艾墓尔欣 只属于神世界的名字。名字来自于 Emerge
  • 所谓的 神阶 只不过是众神在 神式 里的排序。
  • 伪神也是神阶的一类，但并不会算神阶的阶级里
  • 不愿进入 神式 的其他神明，不会受到 众神的世界式 的影响。及会因信仰而消失，因愿望而诞生。
  • 他界的神明可以受第一阶级、第二阶级的神 某伪神 邀约成为此界一员。他界的神明也可向神界申请成为 Emerge 的一员。受不受理是另一回事
  • Emerge 的掌权者是第一阶神的某人，只有其余第一阶神和部分第二阶神知道是谁。
  • 待补充

# 吸血鬼

能在太阳下存在的吸血鬼

# 黄昏

  • 蒲木泽夜
  • 极少数

# 观月

  • Rosa
  • Unconscious
  • 待补充
  • 常年的自然选择，如今的吸血鬼已经可以在太阳在存在。
  • 和人类和睦共处可能吧，互相遵守各自的底线。
  • 吸血鬼受伤只要吸血就能得到强大的自愈术，吸血鬼之间的吸血没有任何作用。
  • 禁止在拒绝的情况下吸血
  • 不会一次性吸干。被吸血的时候会产生一种激素会使人全身麻醉，分泌大量肾上腺素
  • 十个人当中就有一到两个人是吸血鬼。
  • 黎明、正午、黄昏的时候，能力是平常的三分之一
  • 待补充

# 幽灵

存在于彼岸的种族

# 虚灵

  • 七泉夜澪

# 死灵

  • 待补充
  • 虚灵是生物死后，无法转生、转世，且没有怨念的形态。
  • 虚灵也可以被制造出来。
  • 死后有强烈怨念，无法转生、转世的灵魂会转变为死灵。但一般会被死神收走
  • 幽灵是没有重量的，但拥有强大能力的可以实现身体的 实体化 。
  • 拥有 附身 的天生能力。
  • 怕盐，也有特例。
  • 待补充

# 其他

  • 埃斯库・爱丽丝
  • 谷雨
  • 五月雅
  • 悠六峰歌
  • 本质是人类。但也可以拥有特殊能力
  • 一般的特殊能力比不上其他的种族。但经过常年的训练可以变得强大。
  • 拥有与生俱来非常强的能力。
  • 人的能力是有限的，但期望是无限的。
  • 神诞生于人的愿望。
  • 契约、强大的魔力可以实现延长寿命。人终有一死
