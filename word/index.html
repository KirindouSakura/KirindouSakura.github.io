# 世界观

  • 只属于神的世界与世界式。

  • 神不再因信仰而消逝，不再因愿望而诞生的世界。

  • 无数 虚无的世界 ，因拥有了 世界式 而诞生了生物，从而拥有了人、物、妖、神、幽灵、吸血鬼等。

  • 世界式 只是一个概念，却真实存在。

  • 如今再探索虚无的世界已经几乎不可能了，创造新世界则是需要几乎能 书写世界式 的力量。

神的世界式
  • ❀ 神界 - Emerge
    • 神不再因信仰不足而消失
    • 不会再因愿望而诞生，神与神之间可以生神。
    • 只属于神的世界。
    • 不再受外界对神的所有束缚。
    • 神的能力可以为自己所用。
    • 神不再是彼岸边缘的存在。
    • 正在模仿 人类 的生活方式。机械、农田、实验、交易、阶级等 因为是神，在很多方面没有人类的烦恼。

# 世界式

  • 每个世界都存在属于那个世界的世界式，俗称 Rule。
  • 每个世界的世界式有一定区别。比如，有些世界可以实现自由交配同性恋，而其他世界没有这样的规则。
  • 世界式可以改写，但大量改写世界式的代价不仅仅是死这么简单。添加世界式的规则所支付的代价很小。能触碰世界式的人肯定都很强大
  • 一旦出现与此界相矛盾的「异物」，世界式会选择 排斥 。能 触碰 世界式的人可以决定「异物」的存在。
  • 世界式的排斥可以是死亡、存在清除、传至彼岸。
  • 创造的新世界也可以由 创世者 制定世界式，但世界的容量却很有限。复制世界可以提升世界的容量，但部分世界式已经被制定下来。即使是神也无法创造出可以容下不定变量因素的新世界。拥有能力者的世界
  • 一旦制定完毕，即使是创世者也无法轻易重新制定。

# 词汇解释

# 彼岸

  • 活着的生物，与死后的世界所必须经过的地方。
  • 天堂 与 地狱 的交界线。
  • 彼岸 的地方长满了 曼珠沙华 。会使死者产生幻觉，看到最想看到的人、物、事。临死前看到的 走马灯 就来自于彼岸的力量。
  • 徘徊于 彼岸 的人，无法转生。
  • 神是典型能够来回彼岸的种族。所以现实的生物无法看到神的存在，神能靠自己的能力显现。部分神无法从脱离彼岸的能力，即无法显现在现实。出入彼岸与来回彼岸不同
  • 死者无法长久停留在彼岸。
  • 三途河毕竟之地。

# 白观妖

  • 原本是「天羅」神社的一颗普通的樱花树。
  • 一年四季都茂盛的大樱花树。 鸟居内 的樱花树都受它的影响，不会凋谢。
  • 在神社换主后，成为了 大妖怪 ，能够吸引人们来祈祷许愿，代价是 灵魂 。现一直在神社内，与外界相连则需要外界的樱花树绑上「结缘绳」。
  • 主人是「祈愿容器」
  • 转换的祈愿的能量将传输给容器
  • 待补充

# 银瞳

  • 已经 死过 的人或妖，无论以任何方式重生 (转世、转生除外)，瞳孔的眼睛会变成银色。
  • 转世之人 取回记忆，瞳孔颜色会逐渐变成银瞳，代表逐渐接近死亡。
  • 象征经历过死亡。
  • 相比 天生银瞳 的人 / 妖，颜色会相对更淡、更浅。

# 重生、转世、转生

  • 重生指死后在彼岸的时候，被拉回现实，成为 其他的存在 。
    • 保留了生前的 记忆、样貌、灵魂 。
    • 能力可能会发生 改变、扭曲 。契约的关系可能会 彻底改变能力 ，或者 增加能力 。
    • 重生必须以 某人、某媒介 的力量，是否 彼岸 的存在有重大关系。
  • 转世指死后灵魂被分配到了其他 世界 。可以是原本世界，不同的种族，也可以的他界。
    • 遗失了生前的全部记忆，改变了样貌，保留的 只有灵魂 。
    • 通过 非法手段 可以取回前世的记忆，代价是 肉体可能会无法承受过多的记忆 ，而快速死亡。特殊种族除外
    • 前世的契约可代入今世，记忆部分只会是碎片的形式复现于脑海中。强制恢复记忆会导致身体无法承受过多的记忆。
  • 转生指死后经过彼岸后，通过阎魔的筛选后才能回到现实，一般是 污秽的灵魂 。
    • 失去了生前的全部记忆、样貌发生了改变、 灵魂也被重制 。
    • 无法取回生前的记忆，通过 非法手段 取回的记忆，可能导致 灵魂破损、破坏 。

# 灵魂「たまし」 - Soul

  • 灵魂的破坏 无法 转生、转世、重生。灵魂献于 白观妖 后，也无法转生、转世、重生。
  • 世界上的任何生物，都拥有灵魂。（死灵除外）
  • 神的灵魂是 神髓 。
  • 灵魂的颜色会根据个人情况，而显现出 不同的颜色 （可发生改变，属于正常现象）。刚诞生的灵魂均是 纯色 ，被染上 污秽 (灵魂四周会显现黑色的杂质) 后无法清洗，只有「阎魔」能重制。
  • 灵魂是任何生物最重要的东西。

# 时空间守则 - SpacetimeRule

出现了不同的时间线后，拥有了所谓的平行世界。而对于不同时间、不同空间平行世界的时间线，观测者则必须选择及遵守以下条规。
以时空穿越的「人」为 观测者 ，而与之相关的人为 关联者 ，其余人则是 无关者

# 「α」时间线 - Alpha

  • 首先在 Alpha 的 观测者 回到过去的时间里。
  • 改变过去某一事件的转折点。
  • 未来因过去的变化不会发生 蝴蝶效应 。未来的关联者会因过去发生改变，更新原有的记忆。即：拥有两份记忆。

Alpha 时间线主要是用于改变对应 关联者 的记忆。未来不会发生任何改变。

详细Rule
  1. 观测者 所改变的 关联者 的过去，不能引起同时间人的观测。否则会影响他人成为 新的关联者
  2. 过去的时间里，观测者不能做出改变其他事件的关键点动作。
  3. 禁止被过去 观测者 察觉。即：观测者禁止被过去的自己发现。
  4. 观测者 时间有限。无法存在 24 小时以上。
  5. 禁止对 关联者 、 无关者 表明自己的身份。

# 「β」时间线 - Beta

  • 首先在 Beta 的 观测者 回到过去的时间里。
  • 改变过去发生的事件。
  • 未来因过去的变化而发生巨大的改变。未来的关联者不会留下改变前的记忆。即：只拥有改变后的记忆。

Beta 时间线主要用于改变过去发生的事件，从而改变未来。

详细Rule
  1. 观测者 所改变的过去，过去的所有人均是 关联者 。
  2. 关联者 所经历的事将发生改变。
  3. 禁止被过去 观测者 察觉。即：观测者禁止被过去的自己发现。
  4. 被过去的 观测者 察觉会强制拉回原来的时间， 观测者 的记忆因此更新。
  5. 观测者 的存在时间会因个人体质发生改变。24h 以上则需要付出代价。

# 「γ」「λ」时间线 - Gamma、 Lambda

  • 首先在 Gamma 的 观测者 回到在时间 （Lambda） 里
  • 观测者 记录发生的事件。
  • Gamma 会因 Lambda 的所做的事情发生改变，而回到 Gamma 加以改进。
    PS： Gamma 观测者与自身只存在一个， Lambda 则观测者与自身可同时存在。

Gamma、 Lambda 主要用于预测未来的事件，或者轮回者的通道。

详细Rule
  1. 无关联者 。两条不相通却相同的世界。
  2. Gamma 的观测者取代的是自己。 Lambda 的观测者可与自身相遇。
  3. Lambda 不稳定，会因外界的影响，而改变过去、未来。
  4. 因无 关联者 ，记忆也无法更新或改变。
  5. Gamma 是轮回者经常使用的时间线。但轮回者可能会因反复修改的变量，从而出现 无量者 。无量者即世界以自己为中心无限轮回，从而魔力发生无量变化。

# 「δ」时间线 - Delta

  • Delta 会吞噬 观测者 、吞噬 观测者 时间的危险空间。 观测者 则停留于虚无空间（没有任何时间流动以及空间流动的地方）。
  • 关联者、无关者、无量者都存在于这个时间线之中。
  • Delta 的世界将开始加速。
  • 拥有改变历史动向、未来变量的惩罚。
  • 一般时空管理者都会将所有世界的时间线都转向 Delta 。目的是为了有心人无限改变世界的历史。

最危险却最常用的条规。

详细Rule
  1. 观测者 必须承受改变历史动向、未来变量带来的惩罚。改变越大，惩罚越严重。
  2. 改变的事件， 关联者 、 无量者 无法更新清楚的记忆。记忆会以梦、幻觉、自我感知等虚幻的形式提现。
  3. 无关者 无法拥有相关改变的记忆。其若拥有可更新不同时间线的 能力者 ，则将付出相应的代价。代价一般不会比 观测者 严重。
  4. 无量者 拥有获得所有时间线记忆的权利，但代价是自身变成彼岸存在。
  5. 观测者 的身份一旦被揭穿，时间线将通过其他形式弥补所谓「不属于这个世界」的异物所带来的变量。
  6. 大概率会触发 蝴蝶效应 。

PS：随意改变时空间均为禁忌。

# 虚无空间

  • 停留于虚无空间的人称为：时空流浪者。
    • 没有时空间管理者的允许，无法回到原来世界。
    • 可以看到他人穿梭。
  • 观测者 在虚无空间停留时间最大为 10min
    • 一般停留时间为 3~5s
    • 两边的时间线不会改变。
  • 无任何时间、空间流动概念的地方。
    • 无尽空间大小。
    • 没有时间概念，自身不会变老，也不会感觉到任何新陈代谢。

# 回灵

  • 使自身魔力释放手段，可以达到平常魔力浓度的 3~5 倍。
  • 有部分妖、灵、魔会使自己变成一般形态，回灵则是恢复原姿态。有部分却回灵只是恢复姿态，不会增强魔力浓度。
  • 并不是每个妖、灵、魔都有回灵，需要大量时间修炼。

# 黄泉

  • 来自黄泉的女王「伊邪那美」是现在统治者。
  • 黄泉不等于地狱，但地狱也是黄泉的一部分，三途河的引路人 ——「信薇儿」，住在三途河岸的「彼岸」。
  • 死神均是「伊邪那美」的下属。死神都来自黄泉，可以自如来往地狱、彼岸、黄泉。
  • 「伊邪那美」利用「祈愿」把地狱和黄泉分开，风穴概念消失。去往黄泉必须经过三途河后到达地狱，经过允许才能到达黄泉。
  • 「黄泉」这个概念已经成为世界的一个名字。拥有独立的世界式，书写黄泉世界式的正是「伊邪那美」。
  • 与神界属于两个完全不同的世界。
