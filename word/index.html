# 世界观

  • 只属于神的世界与世界式。

  • 神不再因信仰而消逝，不再因愿望而诞生的世界。

  • 无数 虚无的世界 ，因拥有了 世界式 而诞生了生物，从而拥有了人、物、妖、神、幽灵、吸血鬼等。

  • 世界式 只是一个概念，却真实存在。

  • 如今再探索虚无的世界已经几乎不可能了，创造新世界则是需要几乎能 书写世界式 的力量。

神的世界式
  • 神界 - Emerge
  • 神不再因信仰不足而消失
  • 不会再因愿望而诞生，神与神之间可以生神。
  • 只属于神的世界。
  • 不再受外界对神的所有束缚。
  • 神的能力可以为自己所用。
  • 神不再是彼岸边缘的存在。
  • 正在模仿 人类 的生活方式。机械、农田、实验、交易、阶级等 因为是神，在很多方面没有人类的烦恼。

# 世界式

  • 每个世界都存在属于那个世界的世界式，俗称 Rule。
  • 每个世界的世界式有一定区别。比如，有些世界可以实现自由交配同性恋，而其他世界没有这样的规则。
  • 世界式可以改写，但大量改写世界式的代价不仅仅是死这么简单。添加世界式的规则所支付的代价很小。能触碰世界式的人肯定都很强大
  • 一旦出现与此界相矛盾的「异物」，世界式会选择 排斥 。能 触碰 世界式的人可以决定「异物」的存在。
  • 世界式的排斥可以是死亡、存在清除、传至彼岸。
  • 创造的新世界也可以由 创世者 制定世界式，但世界的容量却很有限。复制世界可以提升世界的容量，但部分世界式已经被制定下来。即使是神也无法创造出可以容下不定变量因素的新世界。拥有能力者的世界
  • 一旦制定完毕，即使是创世者也无法轻易重新制定。
